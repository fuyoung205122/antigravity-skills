# 投研团队：四角色并行分析框架

# 功能說明
自動化執行多Agent并行投資研究分析，產出深度投資報告。

# 解決什麼問題
傳統的單一AI或人工投資研究，在處理複雜數據、多角度分析和驗證方面存在效率低、深度不足、易有偏見等問題。本 Skill 透過模擬專業投研團隊，利用多Agent并行協作，提高研究效率和報告質量。

# 適合什麼情境
1.  對單一上市公司進行深度盡職調查。
2.  評估潛在的投資機會，需要快速且全面的行業及公司分析。
3.  需要驗證現有研究報告的準確性和完整性。
4.  作為投資決策前的輔助參考，提供多維度分析視角。

# 如何觸發
範例1：
對「騰訊控股」進行深度投資研究。

範例2：
請以「台積電」為例，啟動四角色的投研團隊進行全面分析。

範例3：
幫我分析「蘋果公司」的投資價值，啟動AI投研團隊。

範例4：
啟動投研團隊，研究「阿里巴巴」的商業模式、財務狀況、行業地位及風險。

範例5：
用投資研究團隊框架，評估「特斯拉」是否值得投資。

# 輸入內容
需要分析的公司名稱。
例如：騰訊控股、台積電、蘋果公司、阿里巴巴、特斯拉。

# 輸出內容
一份結構化的深度投資研究報告，包含：
1.  一句話結論
2.  四維評分總表
3.  核心數據速覽
4.  各維度分析摘要
5.  投資論點（看多/看空）
6.  巴菲特買入前Checklist
7.  最終投資建議（包含定性判斷、操作建議、催化劑）
8.  總結段落
9.  一份以 `~/{公司名}投資研究報告_{日期}.md` 命名的本地檔案。

# 注意事項
1.  **并行啟動**：所有4個Agent必須在同一條訊息中并行調用。
2.  **訊息通信**：Agent之間通過SendMessage 進行報告，而非文件協作。
3.  **數據準確性**：關鍵數據會要求Agent從多個來源搜索並進行交叉驗證，並標記來源。
4.  **結論明確性**：報告會提供明確的投資建議和價格區間。
5.  **數據支撐**：所有分析均需有數據和來源支持。
6.  **耐心等待**：Agent的研究需要時間，請耐心等待並實時關注進度更新。
7.  **反偏見與局限性**：會對AI研究的潛在偏見和局限性進行說明。
8.  **誠實原則**：在信息稀缺時，報告會明確標記「數據不足」，避免虛假確定性。

# 常見問題
**Q1：如果某個Agent的研究過程中出現問題怎麼辦？**
A1：Skill 會嘗試處理，但若遇到嚴重問題（如無法獲取關鍵數據），可能會影響報告的完整性，並在報告中標記。

**Q2：AI輸出的評分和建議是絕對的嗎？**
A2：AI輸出的評分和建議是基於公開數據和分析框架的結果，僅供參考，不構成投資建議。最終投資決策需由使用者自行判斷。

**Q3：報告中引用的數據來源是否完全可靠？**
A3：Skill 要求Agent從公開的、可靠的金融數據網站和財報中獲取數據，並盡可能交叉驗證。但仍建議使用者自行核實關鍵數據。

**Q4：研究過程需要多久？**
A4：通常需要幾分鐘，具體時間取決於公司信息豐富度、網絡狀況以及Agent的處理速度。
