# Investment Memo Craft

## 功能說明
將投資研究轉化為一份決策就緒的 Codex 研究報告。

## 解決什麼問題
AI 生成的投資研究報告通常缺乏投資者所需的實質性分析、清晰的決策指引和簡潔的格式。此 Skill 旨在解決此問題，使報告更具金融嚴謹性、易讀性，並包含反向分析、估值到行動的指導、投資者專屬建議、克制的排版以及明確的買入/持有/賣出信號。

## 適合什麼情境
*   撰寫、重寫、修改或評論公司/行業/基金的研究報告，特別是需要金融嚴謹性、可讀性、反向分析、估值到行動的指導、投資者專屬建議、克制排版和明確買入/持有/賣出信號的長篇 Markdown 報告。
*   將自動生成的投資研究內容格式化為一份決策就緒的報告。
*   當需要對已有的投資研究進行結構化和決策性改進時。

## 如何觸發

範例1：
當我使用 `investment-research` Skill 生成一份關於蘋果公司 (AAPL) 的研究報告後，使用此 Skill 來優化其格式和內容。

範例2：
請使用 Investment Memo Craft Skill 來改寫這份關於電動車行業的分析報告，使其更適合投資者決策。

範例3：
我需要一份關於 [某公司名稱] 的詳細投資報告，請應用 Investment Memo Craft 的風格和結構。

範例4：
使用 Investment Memo Craft Skill 來分析這份關於 [某基金名稱] 的研究，並提供買入/持有/賣出建議。

範例5：
對此份長篇 Markdown 格式的 [某行業] 研究報告，應用 Investment Memo Craft 的排版和內容標準。

## 輸入內容
*   **原始研究報告內容**：可以是文字、Markdown 格式的研究報告，或是其他形式的投資分析資訊。
*   **公司/行業/基金名稱及相關數據**（非必需，但有助於提升準確性）：如研究日期、股價、市值、估值倍數等。
*   **使用者明確要求**：例如，是否需要「投資備忘錄」格式，或是有特定的關注點。

## 輸出內容
*   **結構化和格式化的研究報告**：遵循「Investment Memo Craft」的排版和內容標準，包括：
    *   AI 研究偏見自覺
    *   核心數據總覽
    *   生意本質分析
    *   護城河評估
    *   逆向思考與風險清單
    *   管理層評估
    *   行業與文明趨勢
    *   估值與安全邊際
    *   最終決策與行動清單
    *   AI 分析置信度 vs 投資確定性
    *   數據來源與審計記錄
*   **增強的分析內容**：包含更清晰的商業機制、反向思考、機會成本、行動閾值、投資者專屬建議等。
*   **克制的排版**：使用 Markdown 語法，提供易於閱讀的層級結構和視覺引導。
*   **明確的買入/持有/賣出信號**：基於分析結果，提供具體的行動建議。

## 注意事項
*   此 Skill 主要用於 Codex 模型，不應直接修改 Claude Code 的 slash-command 來源。
*   不適用於修改 Claude Code slash-command 來源。
*   此 Skill 是寫入式和判斷式疊加，不取代財務數據規則、第一手資料檢查、估值工具或報告審計工具。
*   對於長篇 AI Berkshire 輸出，預設標題為「研究報告」，除非用戶明確要求「投資備忘錄」格式。

## 常見問題
**Q1: 如果我有一個現有的研究報告，是否可以直接貼上來讓 Skill 進行優化？**
A1: 是的，您可以提供現有的研究報告內容，Skill 將會根據其標準進行重寫、結構化和優化。

**Q2: 這個 Skill 生成的報告是否可以直接用於實際投資決策？**
A2: 此 Skill 旨在將報告轉化為更易於決策的格式，但最終的投資決策仍需結合您自身的判斷和風險承受能力。Skill 提供了決策所需的資訊和結構，但不代表保證回報。

**Q3: Skill 支援哪些語言的報告？**
A3: 原始 Skill 內容主要以英文描述，但其結構和分析框架是通用的。中文的範例輸出顯示了對中文報告的支援。

**Q4: 我可以自定義報告的某些部分嗎？**
A4: Skill 的設計是遵循一套標準流程，但您可以通過明確的指令來引導其生成特定的內容，例如要求加入特定的分析維度或調整某些部分的優先級。
