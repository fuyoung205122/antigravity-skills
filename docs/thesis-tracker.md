# Thesis Tracker (投资论文追踪)

## 功能说明
AI 驱动的投资论文追踪系统，帮助投资者在买入后进行系统化、纪律性的跟踪和评估。

## 解决什么问题
1.  **买入后的系统化跟踪缺失**：许多投资者在买入后缺乏持续跟踪机制。
2.  **卖出时机把握不当**：导致该卖时舍不得卖，不该卖时恐慌卖出。
3.  **遗忘买入初心**：随着时间推移，忘记了最初的买入理由和逻辑。

## 适合什么情境
*   **股票投资**：尤其是中长期价值投资。
*   **买入前**：希望为买入决策建立清晰的跟踪和退出机制。
*   **买入后**：需要一个系统化的流程来定期审视投资，确保决策的有效性。
*   **应对市场波动**：在市场波动时，提供客观的参考依据，避免情绪化决策。

## 如何触发
1.  **建立新投资论文**：
    *   `请帮我建立 [公司名] 的投资论文`
    *   `[公司名] 建立论文`
    *   `我想为 [公司名] 跟踪买入`
2.  **追踪检查现有投资论文**：
    *   `请检查 [公司名] 的投资论文`
    *   `[公司名] 季度检查`
    *   `更新 [公司名] 的持股跟踪`
    *   `[公司名] 检查我的买入逻辑`
    *   `对 [公司名] 进行买入后评估`

## 輸入內容
*   **公司名**：需要追踪的公司名称。
*   **操作指令**（可选）：例如“建立论文”或“季度检查”。

## 輸出內容
*   **如果建立新论文**：
    *   一份包含核心论文、核心假设、红线清单、估值锚点的投资论文文件 (`reports/{公司名}-thesis.md`)。
*   **如果进行追踪检查**：
    *   一份详细的追踪报告，包含：
        *   论文健康度评分
        *   核心假设检查结果（含状态更新）
        *   红线检查结果
        *   本期关键变化摘要
        *   估值更新
        *   结论与行动建议（加仓/持有/减仓/清仓）
        *   下次检查重点
    *   更新后的投资论文文件，包含本次检查记录。

## 注意事項
*   **文件路径**：Skill 默认会将论文保存在 `reports/` 目录下。如果 Skill 找不到现有论文，可能会询问文件路径。
*   **数据准确性**：依赖于 WebSearch 获取的数据，建议用户在关键决策时自行交叉验证。
*   **主观性**：部分“核心论文”和“红线条件”的设定仍带有一定主观性，需要用户根据自身理解来定义。
*   **不提供投资建议**：Skill 仅提供分析工具和结构化报告，最终决策仍由用户自行负责。

## 常見問題
*   **Q: 如果我找不到 [公司名]-thesis.md 文件怎么办？**
    A: Skill 会检测到文件不存在，并引导您进入“建立论文”模式。如果 Skill 误判，您可以在后续对话中明确告知 Skill 文件已存在，并可能需要提供文件路径。
*   **Q: 我可以自定义“核心假设”和“红线条件”吗？**
    A: Skill 的设计允许用户在建立论文时自定义这些内容。后续的追踪会基于您最初设定的这些条件进行。
*   **Q: “论文健康度评分”是怎么计算的？**
    A: 评分基于核心假设的状态（破裂、受损、弱化、成立）以及是否触发红线条件。具体公式和标准在 Skill 的详细说明中有解释。
*   **Q: Skill 会帮我自动执行买入或卖出操作吗？**
    A: 不会。Skill 仅提供信息分析和决策辅助，不执行任何交易操作。
