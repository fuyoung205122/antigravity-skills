# SkillsMP 每日熱門趨勢總報告生成技能 (skillsmp-daily-report)

# 功能說明

透過多代理人 (Multi-agent) 同步分析技術，自動彙整 SkillsMP 平台上的每日熱門技能、社群動態與開發趨勢，並產出專業的市場洞察報告。

# 解決什麼問題

解決開發者與投資者在海量 Agent Skill 中迷失、難以即時追蹤熱門項目、無法快速掌握社群評價，以及缺乏開發靈感等痛點。透過此技能，使用者無需手動翻閱多個平台（如 GitHub, Reddit, X），即可獲得結構化的生態系分析。

# 適合什麼情境

1. **每日例行追蹤**：開發者晨間了解今日市場風向。
2. **開發選題決策**：當不確定要開發什麼 Skill 時，獲取「To-C/To-B/To-D」方向建議。
3. **競品調研**：分析現有高採用率技能的撰寫結構與技術棧。
4. **社群口碑觀測**：快速掌握開發者對特定技術或 API 的真實反饋。
5. **神級工具發掘**：尋找能提升工作效率的「必裝」技能清單。

# 如何觸發

請輸入以下指令或類似描述：

* **範例 1**：Skills每日熱門趨勢總報告
* **範例 2**：分析今天的 SkillsMP 趨勢並生成報告。
* **範例 3**：現在 SkillsMP 上最熱門的 Agent Skill 有哪些？
* **範例 4**：幫我跑一次技能趨勢分析，我想找開發靈感。
* **範例 5**：執行 skillsmp-daily-report 任務。

# 輸入內容

* **核心指令**：輸入「Skills每日熱門趨勢總報告」或相關分析要求。
* **可選參數**：可指定特定時間區間（如：本週、今日）或特定類別（如：DevOps 相關）。

# 輸出內容

系統將生成一份名為 `skillsmp_daily_briefing.md` 的專業 Markdown 報告，包含：
1. **核心市場動態**：當前 Agent 生態的整體概況。
2. **爆紅與高潛力排行榜**：列出熱門技能、作者及數據（附網路連結）。
3. **生態雷達**：GitHub、Reddit 等社群的最新專案與討論（附網路連結）。
4. **未來爆紅預測**：預判下一個可能熱門的自動化主題。
5. **開發 Action Items**：建議優先投入的開發方向與變現機會。

# 注意事項

1. **即時性限制**：報告內容取決於搜尋當下的網路資料與 API 抓取結果。
2. **網路依賴**：本技能需在可連接外部網路（GitHub, X, Reddit 等）的環境下執行。
3. **連結有效性**：報告中提供的連結為第三方平台內容，其有效性視該平台而定。

# 常見問題

* **Q：這份報告多久更新一次？**
  * A：此技能為手動觸發，建議每日或每週執行一次以獲取最新動態。
* **Q：為什麼有些技能沒有出現在排行榜？**
  * A：排行榜是基於成長速度、下載量與社群熱度綜合運算的結果，若技能為剛發布且未引起討論，可能暫未入榜。
* **Q：我可以指定只分析特定類別的技能嗎？**
  * A：可以在觸發時加上限制條件，例如「分析 SkillsMP 今日關於 Web Scraping 的熱門趨勢」。
