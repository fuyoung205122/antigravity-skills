# 去劣筛选

# 功能說明

使用7个关键财务指标和3条豁免规则，快速排除不符合一流公司标准的上市公司。

# 解決什麼問題

在海量公司中，快速识别出明显不具备"一流"潜质的公司，节省进一步深入研究的时间和精力。

# 適合什麼情境

*   **初步筛选股票池**：在投资前，快速剔除非优质标的。
*   **行业/市场研究**：快速评估一个行业或市场的整体上市公司质量。
*   **主题投资梳理**：在热门主题下，快速筛选出基本面扎实的潜在投资标的。
*   **风险控制**：避免投资于基本面存在严重缺陷的公司。

# 如何觸發

範例1：
`请帮我筛选一下腾讯、美团、英伟达这几家公司。`

範例2：
`对中国啤酒行业的主要上市公司进行去劣筛选。`

範例3：
`请对恒生指数的成分股进行去劣筛选。`

範例4：
`帮我筛选一下全球云计算行业的上市公司。`

範例5：
`对沪深300指数的成分股进行质量筛选。`

# 輸入內容

*   **公司名称/代码**：可以直接输入一个或多个公司名称或股票代码。
*   **行业名称**：输入行业名称，Skill 会自动搜索该行业的主要上市公司。
*   **市场/指数名称**：输入市场或指数名称，Skill 会自动拉取成分股列表。
*   **主题名称**：输入投资主题，Skill 会搜索相关公司进行筛选。

# 輸出內容

Skill 输出将包含：
*   **汇总表**：清晰展示每家公司的筛选结果（通过、排除、豁免通过）。
*   **通过的公司列表**：列出所有筛选通过的公司。
*   **排除的公司列表**：列出被排除的公司，包含触犯的指标、具体数据和排除理由。
*   **豁免通过的公司列表**：列出因满足豁免规则而通过的公司，包含豁免条款、具体数据和豁免理由。
*   **边界争议说明**：对接近阈值但未明确通过或排除的公司进行补充说明。
*   **板块总结（仅限行业/市场/主题模式）**：包含通过率统计、行业质量判断，以及对行业内最值得关注公司的推荐。

# 注意事項

1.  **银行/保险公司**：第3条（利息覆盖倍数）不适用，因为其商业模式是基于利差经营。
2.  **REITs**：ROE 可能因物业重估而波动，建议使用"核心营运利润ROE"作为替代。
3.  **数据缺失**：如果某项关键数据无法获取，将标注为"数据不足"，而非直接判定通过或排除。
4.  **周期性行业**：应使用完整经济周期的平均值来评估指标，而非单一年份的数据。
5.  **上市时间短**：对于上市不足5年的公司，将使用所有可得数据进行评估，并在结果中标注"数据窗口不足"。
6.  **局限性**：通过此筛选不代表公司"一定好"，仅能排除"明显不好"的公司。通过的公司仍需进行更深入的商业模式、管理层、估值及竞争格局研究。

# 常見問題

*   **Q1：为什么银行和保险公司不适用利息覆盖倍数指标？**
    A1：银行和保险公司的核心业务就是管理资金的利息差，其负债结构与普通公司不同，因此"利息覆盖倍数"的计算和解读方式不适用。

*   **Q2：如果一家公司上市时间很短，比如只有2年，会怎么处理？**
    A2：Skill 会使用这两年所有可得的数据进行分析，但会在结果中标注"数据窗口不足"，提示分析师该结果的局限性，需要结合其他信息进行判断。

*   **Q3：什么是“自由现金流为负”？这代表什么？**
    A3：自由现金流（FCF）是指公司在扣除维持运营和增长所需资本性支出后，剩余的现金流。自由现金流为负意味着公司在维持现有运营和扩张上花费的现金多于其产生的经营现金，可能意味着公司在烧钱，或者投资于扩张但尚未产生回报。

*   **Q4：“总股本膨胀>20%（非并购原因）”是什么意思？**
    A4：这个指标衡量的是公司在过去5年内，其总股本的增长比例。如果增长超过20%且不是因为明确的并购行为，则可能意味着公司通过增发股票等方式稀释了现有股东的权益，这通常是一个负面信号。

*   **Q5：什么是“豁免规则”？什么时候会用到？**
    A5：豁免规则是为了避免"宁可漏网不可误杀"的原则，针对某些特定情况下的公司，即使某个指标不达标，但基于其商业模式或其他强劲信号，仍然认为其具备一流公司的潜质。例如，处于战略投入期、高毛利率但主动低利润率、或高周转薄利模式的公司，可能通过豁免规则而获得通过。
