# PDF Processing Skill

# 功能說明
處理 PDF 檔案，包含讀取、文字/表格提取、合併、分割、旋轉、浮水印、建立、填寫表單、加密/解密、提取圖片以及 OCR 掃描 PDF。

# 解決什麼問題
使用者經常需要處理 PDF 文件，但傳統上這需要專門的軟體或複雜的操作。此 Skill 提供了一個統一的介面來執行各種 PDF 相關任務，簡化了工作流程。

# 適合什麼情境
* **學術研究**: 提取論文、報告中的文字和表格。
* **行政工作**: 合併多個報告、分割長文件、為文件添加浮水印。
* **文書處理**: 建立新的 PDF 文件、填寫 PDF 表單。
* **數位化**: 對掃描的 PDF 進行 OCR，使其具備搜尋功能。
* **文件管理**: 加密敏感文件、解密已加密的文件。

# 如何觸發

範例1：
"請幫我把 `report_part1.pdf` 和 `report_part2.pdf` 合併成一個檔案。"

範例2：
"從 `invoice_oct.pdf` 中提取所有文字和表格。"

範例3：
"請把 `manual.pdf` 的第一頁旋轉 90 度。"

範例4：
"我要建立一個包含「歡迎使用」文字的 PDF 文件。"

範例5：
"請幫我對 `confidential.pdf` 添加密碼保護，使用者密碼是 'user123'，擁有者密碼是 'owner456'。"

# 輸入內容
* PDF 檔案（可透過上傳或指定檔名）。
* 文字內容（用於建立 PDF）。
* 指令（如：合併、分割、提取、旋轉、浮水印、加密/解密、OCR）。
* 旋轉角度和頁碼（若需要）。
* 浮水印檔案（若需要）。
* 密碼（若需要加密/解密）。
* 表單欄位資訊（若需要填寫表單）。

# 輸出內容
* 處理後的 PDF 檔案。
* 提取的文字或表格資料。
* 成功或失敗的執行結果訊息。

# 注意事項
* 處理大型或複雜的 PDF 檔案可能需要較長時間。
* 某些進階功能（如：填寫 PDF 表單）可能需要額外的參數或參考特定指南。
* OCR 功能的準確性取決於原始 PDF 的掃描品質。

# 常見問題
**Q1: 如何合併多個 PDF 檔案？**
A1: 你可以直接要求 Skill 將指定的 PDF 檔案合併，例如：「請將 `file1.pdf`、`file2.pdf` 和 `file3.pdf` 合併成 `combined.pdf`」。

**Q2: 我可以從 PDF 中提取表格嗎？**
A2: 是的，你可以要求 Skill 提取 PDF 中的表格，例如：「從 `data.pdf` 中提取表格」。Skill 會嘗試將表格內容輸出成結構化的格式。

**Q3: 如何 OCR 掃描的 PDF？**
A3: 你可以要求 Skill 對掃描的 PDF 進行 OCR，例如：「請對 `scanned_document.pdf` 進行 OCR 處理」。

**Q4: Skill 能否填寫 PDF 表單？**
A4: 是的，但這通常需要提供欄位名稱和對應的值。請參考 FORMS.md 文件以獲取詳細說明。
