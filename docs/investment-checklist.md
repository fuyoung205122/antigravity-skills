# 巴菲特价值投资买入前 Checklist

# 功能說明

分析一家公司是否符合巴菲特的价值投资标准。

# 解決什麼問題

幫助投資者在買入股票前，系統性地評估公司的基本面、競爭優勢、管理層、估值和風險，避免做出衝動或錯誤的投資決策，降低投資虧損的風險。

# 適合什麼情境

*   **個人投資者**：在考慮買入任何一檔股票前，作為決策輔助工具。
*   **價值投資者**：需要一個結構化的方法來篩選和評估符合價值投資原則的公司。
*   **風險管理者**：用於快速識別潛在的投資風險，避免將資金投入高風險或不了解的標的。
*   **學習投資**：初學者可以透過此 Checklist 學習如何從基本面分析一家公司。

# 如何觸發

範例1：
分析 蘋果公司

範例2：
巴菲特价值投资买入前 Checklist 腾讯, 茅台, 英伟达

範例3：
NVDA AAPL MSFT

範例4：
分析腾讯控股

範例5：
帮我用巴菲特价值投资买入前 Checklist 分析一下特斯拉

# 輸入內容

*   **公司名稱或股票代碼**：可以是單個或多個公司名稱/代碼，以逗號、頓號或空格分隔。例如："蘋果公司", "AAPL", "腾讯, 茅台, 英伟达", "NVDA AAPL MSFT"。
*   **"巴菲特价值投资买入前 Checklist"**：這個指令詞，用來觸發此 Skill。

# 輸出內容

*   **針對每家公司的詳細分析報告**：
    *   **能力圈評估**：是否能理解公司生意模式。
    *   **好生意評估**：營利能力、估值、增長趨勢、財務健康、競爭格局等。
    *   **護城河評估**：品牌、轉換成本、網絡效應、規模優勢、技術壁壘等。
    *   **管理層評估**：誠實度、資本配置能力、股東利益導向等。
    *   **安全邊際評估**：PE, PB, 股息率, FCF Yield 等估值指標，以及三情景下的估值分析。
    *   **倉位與決策紀律檢查**：排除情緒化決策。
    *   **鏡子測試**：用簡潔的語言總結買入理由。
    *   **快速否決清單檢查結果**。
    *   **核心數據表**：包含關鍵財務數據。
    *   **關鍵風險**：列出3-5條主要風險。
    *   **最終結論**：通過 (✅)、未通過 (❌)、灰色地帶 (❓) 或 N/A。
*   **多公司時的總覽對比表**：列出各公司在六關 Checklist 中的評分及核心結論。
*   **報告輸出位置**：`~/巴菲特Checklist-[公司名或"多公司对比"].md`

# 注意事項

*   **資訊豐富度**：如果公司資訊極度稀缺（C級），AI 可能無法填滿所有表格，會誠實標註"數據不足無法判斷"，並聚焦可驗證的核心問題。
*   **AI 研究偏見**：Skill 會進行"信息豐富度"評級，並提醒使用者注意"共識陷阱"或數據不足的局限性。
*   **硬性否決**：某些情況（如管理層有嚴重誠信問題）會直接導致"否決"。
*   **數據來源**：所有數據將盡力從可靠來源獲取，並標註來源時間。
*   **精確計算**：估值和財務指標將盡量使用工具精確計算。
*   **原則遵守**：嚴格遵守"寧可錯過，不可做錯"、"誠實面對能力圈"、"安全邊際是生命線"、"鏡子測試不可跳過"等關鍵原則。

# 常見問題

**Q1: 如果某家公司在 AI 研究中被評為 C 級（信息極度稀缺），會怎樣？**
A1: Skill 會誠實標註"數據不足無法判斷"，並嘗試聚焦在可驗證的核心問題上，不會勉強填滿表格。這意味著 AI 無法給出一個完整的 Checklist 評估，建議在這種情況下，投資者需要自行補充一手資訊或謹慎考慮。

**Q2: 我輸入的公司名字 AI 識別不出來怎麼辦？**
A2: 請嘗試輸入更標準的公司全稱或其股票代碼。如果仍無法識別，可能是 Skill 資料庫中未收錄該公司，或輸入格式有誤。

**Q3: Checklist 中的評分標準是固定的嗎？**
A3: 是的，Skill 內建了基於巴菲特投資哲學的標準化評分標準（★1-5），旨在提供一個客觀的評估框架。

**Q4: 輸出的報告會包含具體的財務數據嗎？**
A4: 是的，報告會包含關鍵的財務數據（如 ROE、毛利率、負債水平、PE、PB、股息率等），並標註數據來源時間。

**Q5: 如果 Checklist 顯示"未通過"，是否就意味著絕對不能買這家公司？**
A5: "未通過"表示公司在某個或多個關鍵環節未能達到價值投資的標準，存在較高的風險。Skill 的目標是排除壞選擇，"未通過"是強烈建議避免投資的信號。"灰色地帶"則意味著存在爭議點，需要投資者自行權衡。
