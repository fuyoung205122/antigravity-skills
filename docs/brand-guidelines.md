# Skill 名稱

brand-guidelines (Anthropic 品牌風格套用工具)

# 功能說明

自動將 Anthropic 官方的品牌色彩與字體排版套用到任何數位成品（如簡報、圖表、網頁），使其呈現出 Anthropic 專屬的設計質感。

# 解決什麼問題

在製作簡報、設計圖表或建立品牌相關成品時，使用者往往難以精確掌握官方的視覺規範。本 Skill 解決了「手動調整色彩、字體費時」以及「視覺風格不一致」的問題，確保所有產出皆符合 Anthropic 官方的專業設計標準。

# 適合什麼情境

* **簡報製作**：設計投影片、Pitch Deck 或產品報告。
* **企業文件排版**：將一般的文字文檔轉換為具備品牌識別度的排版。
* **圖表與視覺設計**：為資訊圖表、組織圖或流程圖套用官方色彩與強調色。
* **跨平台排版保持**：確保在不同裝置與系統上都能維持一致的字體與色彩。

# 如何觸發

請在對話中使用以下類似指令來觸發此 Skill：

* **範例1**：「請幫我把這份簡報大綱套用 Anthropic 的品牌配色。」
* **範例2**：「我想將這段文字格式化，請套用 brand-guidelines 視覺規範。」
* **範例3**：「請使用 Anthropic 的官方字體（Poppins/Lora）排版這篇報告。」
* **範例4**：「利用 Anthropic 的輔助色（橘、藍、綠）來設計這個產品比較圖表。」
* **範例5**：「幫我把這個 HTML 區塊套用 Anthropic 的官方深淺色與字體樣式。」

# 輸入內容

* **文本內容**：需要進行排版的標題與內文。
* **設計物件**：如簡報檔案、圖表、HTML 結構或需要調整顏色的視覺元素。
* **排版指示**：特定需要強調的重點或階層關係。

# 輸出內容

* **標準品牌色彩套用**：
  * **主色**：深黑（`#141413`）、淺米白（`#faf9f5`）、中灰（`#b0aea5`）、淺灰（`#e8e6dc`）。
  * **輔助強調色**：暖橘（`#d97757`）、天空藍（`#6a9bcc`）、草地綠（`#788c5d`）。
* **專業字體排版**：
  * **大標題（24pt以上）**：套用 Poppins 字體。
  * **內文與本文**：套用 Lora 字體。
* **高質感視覺成品**：輸出具備清晰層次、高易讀性且風格統一的設計物件。

# 注意事項

1. **字體相依性**：為了達到最佳的視覺效果，建議在您的本機系統中預先安裝 **Poppins** 與 **Lora** 字體。若系統未安裝，將自動降級（Fallback）使用 Arial（標題）與 Georgia（內文）。
2. **顏色精準度**：本 Skill 在程式碼層面（如 `python-pptx`）會使用精確的 RGB 數值，但在不同的螢幕顯示器上可能會有輕微色差。

# 常見問題

**Q1：如果我的電腦沒有安裝 Anthropic 的官方字體（Poppins 和 Lora），排版會跑掉嗎？**  
A1：不會。系統內建了智慧降級機制，如果偵測不到官方字體，會自動改用 Arial（標題）與 Georgia（內文）代替，依然能保持良好的可讀性與排版結構。

**Q2：橘、藍、綠這三種輔助色會怎麼被使用？**  
A2：這三種顏色主要用於非文字的形狀、圖表線條或強調背景中。系統會自動輪替使用，避免單一顏色過度搶眼，維持畫面的和諧。

**Q3：這個工具只能用在簡報（PPT）上嗎？**  
A3：不僅限於簡報。任何支援自訂 CSS 樣式、RGB 設色或字體設定的成品（如網頁、PDF、圖表），皆可套用此規格。
