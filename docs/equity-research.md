# Enterprise Multi-Agent Equity Research Platform

# 功能說明

透過啟動 10 個專業領域 AI 研究員進行平行深度搜索，並由中央研究委員會統整產出具備證據等級標示的機構級投資研究報告。

# 解決什麼問題

1.  **資訊過載與單一視角**：解決個人研究時難以同時兼顧財報、籌碼、技術、法規、總經等多維度資訊的問題。
2.  **資訊污染（Cross Contamination）**：透過 10 個獨立 Agent 分別運作，確保各領域分析不互相干擾，維持分析的純粹度。
3.  **缺乏嚴謹度**：傳統 AI 容易產生幻覺，本 Skill 強制執行證據評等制度（FACT/OPINION）與來源優先級過濾。
4.  **分析耗時**：自動化收集、過濾並整合巨量市場資訊，將數小時的研究工作縮短至分鐘級別。

# 適合什麼情境

*   **深度基本面研究**：決定長期持股前的全面性體檢。
*   **重大事件評估**：法說會後、除權息前或重大政策發布後的影響分析。
*   **風險管理避雷**：針對公司管理層誠信、法規風險進行深度挖掘。
*   **多場景投資佈局**：需要了解在不同經濟情境（牛/熊/基本）下目標價位與操作策略時。

# 如何觸發

請在對話中輸入：`equity-research <股票代號> <公司名稱>`

範例1：`equity-research 2330 台積電`

範例2：`equity-research 3052 峯典`

範例3：`equity-research 2454 聯發科`

範例4：`equity-research 2317 鴻海`

範例5：`equity-research 2603 長榮`

# 輸入內容

*   **股票代號**：4 位或更多位數的股票代碼。
*   **公司名稱**：公司的正式或通用簡稱。
*   **市場（可選）**：預設為台股，亦可指定其他市場。

# 輸出內容

一份完整的機構級投資報告，包含：
1.  **執行摘要**：核心投資邏輯。
2.  **10 大領域專案報告**：含財務催化劑、情緒輿情、投資策略、技術指標、總經循環、籌碼資金、法規風險、估值敏感度、產業鏈護城河、管理層質量。
3.  **情境預測**：Bull / Base / Bear 三種預測情境及其機率、目標價。
4.  **投資評等與策略**：明確的買賣評等（Strong Buy to Sell）與短中長期配置建議。
5.  **量化評分卡**：各維度的量化分數與目標價計算方法學。

# 注意事項

*   **資料延遲**：數據依據網路搜尋與資料庫更新頻率而定，非即時報價系統。
*   **誠信否決權**：若法規或管理層誠信被評定為重大風險，最終評等將受到強制下調限制（最高僅能為 HOLD）。
*   **數據缺失**：若查無特定資訊會標示 `DATA NOT AVAILABLE`，絕不編造數據。
*   **免責聲明**：報告僅供參考，不代表投資建議，使用者需自負投資盈虧。

# 常見問題

*   **Q：為什麼產出時間比一般對話久？**
    *   A：因為系統正在同時啟動 10 個不同的 Agent 進行深度資料檢索與交叉驗證，並需經過中央委員會的整合流程。
*   **Q：證據評等 [FACT] 與 [CONSENSUS] 有何區別？**
    *   A：[FACT] 是來自財報或交易所公告的確鑿數據；[CONSENSUS] 是指多數分析師或市場普遍的預期，並非絕對事實。
