# 每日 AI 多工助理總報告

# 功能說明

透過同時啟動 7 個專屬 AI 研究代理（Research Agents）進行並行搜尋，全方位涵蓋新聞、社群、購物、生活與學習資訊，並由總控 Agent 彙整成一份具備行動建議的每日綜合情報。

# 解決什麼問題

1.  **資訊過載與碎片化**：解決現代人需要切換多個 App/網站（如新聞、PTT、Dcard、電商、YouTube）才能掌握生活脈動的痛點。
2.  **單一 AI 廣度不足**：傳統單一對話 AI 難以同時深入處理多個領域的即時檢索，此 Skill 透過「多工並行」提升效率與內容深度。
3.  **缺乏行動指引**：不只提供資訊，更透過總控 Agent 分析優先順序，將資訊轉化為具體的行動清單。

# 適合什麼情境

*   **晨間收報**：早起或通勤時，快速閱覽當天全球大事與社群熱點。
*   **週末規劃**：需要同時獲取旅遊建議、院線電影與購物折扣時。
*   **科技/趨勢追蹤**：快速掃描最新的 AI 工具、生產力技巧與股市熱點。
*   **日常比價**：特定節慶或購物季，想知道各大電商優惠與信用卡回饋。

# 如何觸發

範例1：幫我整理今天的資訊

範例2：今日懶人包

範例3：多 Agent 日報，地點設為台中

範例4：daily report，購物比價請針對「摺疊手機」

範例5：今天有什麼新聞？我只想看資訊整理、社群觀察跟影音整理這三個 Agent

# 輸入內容

*   **基礎指令**（必填）：上述觸發詞之一。
*   **自訂選項**（選填）：
    *   地點：預設台北，可更改。
    *   特定商品：指定購物比價的目標。
    *   興趣偏好：如影音推薦可指定「驚悚片」或「韓劇」。
    *   語言：預設繁體中文，可切換。
    *   Agent 篩選：可指定只啟動部分 Agent。

# 輸出內容

1.  **Markdown 完整報告** (Artifact)：
    *   今日你該知道的 10 件事（交叉分析）。
    *   今日建議行動清單（立即/今天/本週）。
    *   優先順序建議（P0~P4 分級）。
    *   7 大領域詳細報告（含表格、Mermaid 圖表）。
2.  **手機版 .txt 純文字檔**：
    *   適合手機螢幕寬度（40 中文字/行）。
    *   使用樹狀結構（├─、└─）與特殊分隔線（━━━）。
    *   方便複製、轉傳與快速閱讀。

# 注意事項

*   **處理時間**：由於需等待 7 個 Agent 同時完成搜尋，啟動後需稍候片刻（系統會即時更新進度表）。
*   **即時性限制**：搜尋結果受限於當前網路資訊與搜尋引擎索引速度。
*   **非專業建議**：投資、醫療或購物建議僅供參考，決策前請再次核實。

# 常見問題

*   **Q：可以每天固定時間發送嗎？**
    *   A：目前需手動輸入指令觸發，建議將觸發詞設為捷徑。
*   **Q：如果不指定地點，行程規劃會給哪裡的建議？**
    *   A：預設為台北市。
*   **Q：.txt 檔案在哪裡找？**
    *   A：生成完畢後，會存放於您的對話工作區（Workspace）中。
