# PPTX Skill

# 功能說明

處理與 `.pptx` 檔案相關的任何任務，包括建立、讀取、編輯、修改或分割簡報。

# 解決什麼問題

使用者需要建立、分析或修改 PowerPoint 簡報時，需要一個專門的工具來處理 `.pptx` 檔案，以確保格式正確、內容完整並具有專業的視覺設計。

# 適合什麼情境

*   建立新的簡報、提案或演講。
*   讀取、解析或從現有的 `.pptx` 檔案中提取文字內容。
*   編輯、修改或更新現有的簡報。
*   合併或分割簡報檔案。
*   使用模板、佈局、演講者備註或註釋。
*   任何需要操作 `.pptx` 檔案的場景。

# 如何觸發

範例1：
幫我用 "Midnight Executive" 的配色方案建立一個關於新產品發布的簡報。

範例2：
從 "Q3 業績報告.pptx" 中提取所有文字內容。

範例3：
請編輯 "專案提案 v2.pptx"，將第二頁的圖表換成新的數據。

範例4：
我需要將 "議程.pptx" 和 "結論.pptx" 合併成一個檔案。

範例5：
請檢查 "使用者手冊.pptx" 的視覺呈現是否有問題。

# 輸入內容

*   `.pptx` 檔案（用於讀取、編輯、修改、合併、分割）。
*   簡報的主題、內容、配色方案、字體要求等（用於建立新簡報）。
*   現有簡報的修改指示。
*   目標簡報的模板或參考檔案（如果有的話）。

# 輸出內容

*   一個新的 `.pptx` 檔案（建立的簡報）。
*   從 `.pptx` 檔案中提取的文字內容。
*   修改或更新後的 `.pptx` 檔案。
*   合併或分割後的 `.pptx` 檔案。
*   針對簡報視覺設計的建議或修正。

# 注意事項

*   在建立或編輯簡報時，請務必提供清晰的指示和內容。
*   對於視覺設計，建議提供具體的配色方案、字體、佈局和視覺元素的參考。
*   在執行 QA（品質保證）時，務必仔細檢查內容和視覺上的錯誤。
*   對於較複雜的編輯或從頭開始創建，請參考 `editing.md` 和 `pptxgenjs.md` 文件。

# 常見問題

*   **Q：如何從 `.pptx` 檔案中提取文字？**
    A：可以使用 `python -m markitdown presentation.pptx` 命令來提取文字內容。
*   **Q：如何建立一個全新的簡報？**
    A：請參考 `pptxgenjs.md` 文件，並提供您想要的內容、設計風格等詳細資訊。
*   **Q：如何編輯現有的簡報？**
    A：請參考 `editing.md` 文件，並提供具體的修改要求，例如修改文字、圖片、佈局等。
*   **Q：如何確保我創建的簡報在視覺上是專業的？**
    A：請參考「Design Ideas」部分，其中包含了關於配色、佈局、字體和視覺元素的建議。
*   **Q：如何進行視覺檢查？**
    A：建議將簡報轉換為圖片（參見「Converting to Images」），然後使用專門的提示詞進行詳細檢查，或使用子代理進行視覺檢閱。
