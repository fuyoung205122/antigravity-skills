# 公司新闻脉搏

# 功能说明

当公司股价出现异动时，快速查找并归因可能的原因。

# 解决什么问题

当公司股价出现非正常波动时，用户难以在短时间内了解具体原因，也无法判断是否需要进一步深入研究或调整投资策略。

# 适合什么情境

*   公司股价在短时间内出现显著上涨或下跌（例如：单日 ±5%，一周 ±10%）。
*   财报发布后，股价出现异动，希望快速了解市场反应。
*   看到关于某公司的吸引眼球的新闻，但不确定其真实影响。
*   作为投资经理或分析师，需要对持仓或关注的公司股价波动进行快速的情报响应。

# 如何触发

範例1：
解释一下拼多多最近股价大跌 10% 的原因。

範例2：
帮我看看苹果公司股价异动的原因，回溯 14 天。

範例3：
特斯拉今天跌了 5%，有什么新闻可以解释吗？

範例4：
我想知道腾讯最近三天股价的异动是不是因为行业政策变化。

範例5：
小米公司上周股价涨了 8%，最近有什么重要事件吗？

# 輸入內容

*   **公司名**：中文、英文或股票代码均可。
*   **时间窗口** (選填)：偵察新聞的回溯天數，預設為 14 天。
*   **股价异动** (選填)：具體的漲跌幅度與時間，例如"跌 12%/3 天"。
*   **关注侧重** (選填)：公司事件、监管、行业、情绪，預設為四方平均偵察。

# 輸出內容

*   **一句话归因**：用一段話說明股價異動的主因、次因及性質判斷。
*   **完整事件时间线**：按日期倒序合併所有維度的事件，標註來源和與股價異動的關聯度。
*   **异动归因表**：列出可能的解釋、證據、反證、置信度和持續性。
*   **性质判断**：判斷此次異動是價值事件、情緒/技術波動、真因不明還是混合。
*   **各维度侦察摘要**：匯總各偵察維度的關鍵發現及歸因貢獻度。
*   **行动建议**：根據分析結果，提供是否建議觸發投資論文重審、財報研讀、管理層重審、調倉或僅觀察的建議。
*   **接下来 7-30 天的跟踪清单**：列出待披露事件、指標或關鍵觀察信號。
*   **信息缺口声明**：說明本次偵察未解決的疑點、找不到的信息或需要等待的披露事項。
*   **异动主因判断**：結合以上信息，判斷股價異動的主要原因。
*   **是否触发论文重审**：根據異動性質和證據，判斷是否需要重新審視投資論文。

# 注意事項

*   此 Skill 定位為「情報快速響應」，而非深度投研。
*   對於信息稀缺的 C 級公司，可能無法找到足夠信息來解釋股價異動，此結論本身亦有價值。
*   不替用戶做最終的投資決策，僅提供基於信息的分析和建議。
*   所有判斷均基於現有公開信息，並註明信息來源。

# 常見問題

*   **Q: 這個 Skill 的時效性如何？**
    A: 設計目標是在 10-15 分鐘內提供股價異動的初步歸因，以滿足快速響應的需求。
*   **Q: 如果找不到任何能解釋股價異動的原因怎麼辦？**
    A: Skill 會誠實報告「真因不明」，並指出可能的信息缺口，這比硬湊因果關係更有價值。
*   **Q: 这个 Skill 和深度研究 Skill 有什么区别？**
    A: 本 Skill 側重於快速的情報收集和歸因判斷，以確定是否需要進一步的深度研究。深度研究 Skill (如 `/investment-team`) 則會進行更全面、深入的分析。
