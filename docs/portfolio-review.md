# portfolio-review

# 功能說明

對投資組合進行審視與優化。

# 解決什麼問題

投資者在研究公司時，常忽略了組合層面的決策，例如買多少、用什麼錢買、以及與已有持倉的衝突等。本 Skill 旨在提供一個系統性的方法，從組合的整體角度來評估和優化投資。

# 適合什麼情境

*   當你持有股票或其他資產，希望了解其在整體組合中的表現和風險。
*   在考慮買入新資產前，評估其對現有組合的影響。
*   定期進行投資組合的健康檢查和再平衡。
*   想了解如何像巴菲特、李錄等頂級投資者一樣進行組合管理。

# 如何觸發

範例1：
`portfolio-review 腾讯30%, 美团20%, 茅台20%, 英伟达15%, 现金15%`

範例2：
`portfolio-review 腾讯 500股 @480港元, 美团 1000股 @130港元, 茅台 200股 @1800人民币, 英伟达 100股 @700美元, 现金 15%`

範例3：
`portfolio-review 我的持仓` (假設已保存 `reports/portfolio-latest.md`)

範例4：
`portfolio-review 审视我的最新持仓并给出优化建议`

範例5：
`portfolio-review 对我目前持有的资产进行深度分析，包括集中度、相关性和机会成本`

# 輸入內容

*   **持倉列表**： 可以是股票代碼和百分比，或股票名稱、股數、買入價格。
    *   格式範例：`腾讯30%, 美团20%, 茅台20%, 英伟达15%, 现金15%`
    *   格式範例：`腾讯 500股 @480港元, 美团 1000股 @130港元, ...`
*   **自動載入**： 如果輸入 "我的持倉"，Skill 會嘗試載入 `reports/portfolio-latest.md` 文件。

# 輸出內容

*   一份詳細的投資組合審視報告，包含：
    *   **組合概覽**：持倉表格、餅圖描述。
    *   **單倉位體檢**：個別持倉的健康狀態，包括是否還會買入、長期持有是否舒適、買入邏輯是否完整。
    *   **組合分析**：
        *   **集中度**：評估持倉是否過於集中或分散。
        *   **相關性**：識別持倉間的隱性關聯及潛在風險共振。
        *   **機會成本**：分析哪些持倉的預期回報最低，是否應該賣出。
        *   **壓力測試**：預估在不同極端情景下的組合影響和最大回撤。
    *   **調倉建議**：具體的加倉、減倉、清倉、新建倉建議及理由。
    *   **尋找替代標的建議**：引導使用者使用其他工具尋找優質標的。
    *   **現金管理建議**：給出建議的現金佔比及理由。
    *   **總結性結論**：組合整體健康度、最應做的一件事、當前最大風險。
*   更新或創建 `reports/portfolio-latest.md` 文件，包含最新持倉、審視結論、調倉記錄和下次審視提醒。

# 注意事項

*   **數據依賴**：Skill 依賴於外部數據源獲取最新的股價、估值指標、財務數據和分析師預期，數據的準確性和時效性會影響分析結果。
*   **估值模型**：預期年化回報的計算基於特定的估值模型，結果僅供參考。
*   **低置信度標注**：對於信息豐富度較低的持倉，分析結論會標註低置信度。
*   **不推薦個股**：Skill 不直接推薦具體個股，而是引導使用者使用 `/industry-research` 或 `/investment-checklist` 等工具進行系統篩選。
*   **自動化限制**：對於需要複雜邏輯判斷或主觀判斷的決策，仍需要使用者最終確認。

# 常見問題

*   **Q: 我應該多久運行一次這個 Skill？**
    A: 建議每季度審視一次，頻繁交易可能損害長期回報。
*   **Q: 我輸入的持倉信息不全怎麼辦？**
    A: Skill 會盡量解析，但建議提供越詳細的資訊越好，例如是否有成本價、股數等，以便進行更精確的分析。
*   **Q: "我的持倉" 功能在哪裡讀取文件？**
    A: Skill 會在 `reports/portfolio-latest.md` 路徑下尋找您的持倉文件。
*   **Q: 壓力測試的結果是估計值嗎？**
    A: 是的，壓力測試結果基於對持倉屬性和歷史波動的定性+粗估評估，旨在提供一個大致的風險範圍。
*   **Q: 為什麼 Skill 推薦我使用其他工具尋找替代標的？**
    A: 為了確保分析的客觀性和系統性，Skill 鼓勵您使用專門的行業研究或篩選工具來發現潛在的投資機會，而非在本 Skill 中直接產生個股推薦。
