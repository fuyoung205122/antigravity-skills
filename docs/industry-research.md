# 行业投资研究

# 功能说明

对指定行业进行系统化的产业链投资研究，涵盖从投资逻辑验证到投资组合配置的完整流程。

# 解决什么问题

传统行业研究往往局限于单一视角或信息碎片化，难以形成全面、深入的投资洞察。本 Skill 旨在通过系统性的框架，帮助用户识别行业中的投资机会，规避潜在风险，并制定科学的投资组合策略。

# 适合什么情境

*   **新行业/主题投资探索**：当对一个新兴或不熟悉的行业产生兴趣时，需要对其进行全面了解。
*   **深入行业基本面分析**：对已关注的行业进行更深层次的研究，验证投资逻辑，评估公司价值。
*   **构建行业投资组合**：需要为特定行业或主题构建一个分散化的投资组合。
*   **评估行业风险与机会**：希望系统性地识别行业内的潜在风险点和高增长机会。
*   **独立投资决策支持**：作为个人投资者或专业分析师，需要一个结构化的工具来辅助投资决策。

# 如何触发

範例1：
请为新能源汽车行业进行产业链投资研究。

範例2：
对人工智能在医疗健康领域的应用进行产业链全景扫描，并重点分析四大师个股框架。

範例3：
生成一份关于半导体材料行业的投资研究报告，包含产业链梳理、公司分析和投资组合建议。

範例4：
帮我研究一下合成生物学行业，需要验证投资逻辑链，绘制产业链图，并扫描全球上市公司。

範例5：
对机器人行业进行产业链投资研究，重点关注卡脖子环节的头部公司。

# 輸入內容

用户需要提供希望研究的**行业名称**。例如：“新能源汽车”、“半导体”、“生物技术”等。

# 輸出內容

一份结构化的行业投资研究报告，包含：
1.  **投资逻辑链构建与验证**：明确的因果关系和证据。
2.  **产业链全景图绘制**：上游、中游、下游及辅助环节的结构图。
3.  **全球上市公司扫描**：按环节分类，并按确定性分层。
4.  **各环节头部公司四大师分析**：生意本质、护城河、风险、管理层、估值和推荐度。
5.  **行业级风险评估**：系统性风险、历史类比、偏误自查。
6.  **文明趋势判断**：行业的长期发展潜力。
7.  **投资组合配置建议**：核心、卫星、期权仓位建议及买卖信号。
8.  **综合决策备忘录**：行业总评表和四位大师模拟点评。
9.  最终报告将以 Markdown 文件形式保存在 `~/[行业名]产业链投资研究报告.md`。

# 注意事項

*   报告的质量高度依赖于输入的行业名称的明确性。
*   AI研究可能存在偏见，用户需结合自身判断。
*   数据抽检流程需要用户协助完成（见报告中的数据抽检部分）。
*   部分分析内容，如“四大师模拟点评”，是基于AI的模拟，仅供参考。

# 常見問題

**Q1: 如果我想研究一个跨领域的行业，比如“科技+医疗”怎么办？**
A1: 建议将研究拆分进行，先研究“科技在医疗中的应用”，再独立研究“医疗健康”相关子领域，最后再整合分析。或者，提供一个更具体的行业名称，例如“数字医疗”。

**Q2: 报告中提到的“四大师”是指谁？**
A2: 指的是投资界四位极具影响力的思想家：段永平、巴菲特、芒格和李录。报告会模拟他们的分析视角。

**Q3: 报告中的“数据抽检”环节我需要做什么？**
A3: 报告生成后，AI会提供一个数据抽检的流程和指令。您需要根据指令，使用可靠的信源（例如金融数据终端、公司财报等）进行数据验证，并将结果反馈给AI，以完成报告的准出流程。

**Q4: 为什么报告最后会要求写入本地文件？**
A4: 这是为了方便用户查阅和管理生成的详细研究报告。`~/[行业名]产业链投资研究报告.md` 是一个示例路径，实际路径可能会根据您的环境有所调整。

**Q5: 报告中的“文明趋势判断”和“行业级风险评估”有什么作用？**
A5: 这部分旨在提供更长远的视角。文明趋势判断帮助理解行业的长期发展潜力，是否是可持续的范式转移；行业级风险评估则帮助识别和规避潜在的系统性风险，确保投资的稳健性。
