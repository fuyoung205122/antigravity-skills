# Skill 名稱

Theme Factory (主題工廠)

# 功能說明

為 Artifacts（如簡報、文件、報告、HTML 網頁）套用專業的配色方案與字體組合。

# 解決什麼問題

解決文件視覺設計不專業、配色不協調或字體搭配混亂的問題。它能快速為枯燥的內容注入具備一致性與美感的視覺風格，讓使用者無需具備專業設計背景也能產出高品質的視覺作品。

# 適合什麼情境

*   製作商務提案簡報，需要穩重專業的視覺感。
*   設計產品著陸頁 (Landing Page)，需要吸引目光的色彩。
*   撰寫研究報告，需要清晰且具備閱讀性的字體搭配。
*   多份文件需要維持統一的品牌視覺識別。
*   需要根據特定概念（如「極地、復古、科技」）快速生成全新視覺主題。

# 如何觸發

範例1：請幫我目前這份簡報換一個比較專業的主題。

範例2：我想看看有哪些預設的配色主題可以用，請顯示主題展示文件。

範例3：請將「Ocean Depths」主題套用到我剛剛產生的 HTML 頁面上。

範例4：我想要一個「復古未來主義」風格的主題，請幫我建立一個並套用到文件中。

範例5：現有的主題都不適合，請幫我生成一個適合「有機農業」報告的綠色系主題。

# 輸入內容

*   **目標對象**：現有的 Artifact（Slide deck, HTML, Document 等）。
*   **主題選擇**：從 10 種預設主題中挑選，或提供風格描述（關鍵字、氛圍）以生成自定義主題。

# 輸出內容

*   **視覺展示**：顯示 `theme-showcase.pdf` 供使用者預覽。
*   **套用結果**：已更新配色（Hex codes）與字體組合（Headers/Body fonts）的 Artifact。
*   **自定義定義**：若為新建主題，會輸出主題名稱與詳細的色彩/字體規範。

# 注意事項

*   修改前會先請求使用者確認選擇的主題。
*   系統會自動確保文字與背景之間的對比度符合易讀性要求。
*   自定義主題生成需要使用者提供明確的視覺描述。

# 常見問題

**Q：有哪些預設主題可以選？**
A：目前提供 10 種，包括：Ocean Depths (深海)、Sunset Boulevard (日落大道)、Modern Minimalist (現代簡約)、Tech Innovation (科技創新) 等，詳細清單可查看 `theme-showcase.pdf`。

**Q：如果我不喜歡預設的主題怎麼辦？**
A：您可以描述您想要的風格，我會為您即時生成一個全新的自定義主題。

**Q：這個 Skill 會改變我的內容嗎？**
A：不會，它僅會調整 Artifact 的視覺樣式（顏色與字體），不會修改您的文字內容。
