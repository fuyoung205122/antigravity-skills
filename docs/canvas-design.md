# canvas-design

# 功能說明

結合獨特的設計哲學與極致的視覺工藝，為使用者量身打造博物館級的原創靜態視覺藝術、海報與設計作品。

# 解決什麼問題

在 AI 創作時代，許多視覺產出容易顯得廉價、公式化或缺乏靈魂。本 Skill 解決了「如何創作出具有深厚美學底蘊、匠人手作感（Craftsmanship）且不流於俗套的原創藝術」這一難題。它摒棄了傳統的大段文字堆砌，改以空間、色彩、幾何與系統性圖樣進行「空間溝通」，並確保產出 100% 原創，免除版權侵權風險。

# 適合什麼情境

*   **藝術與展覽海報設計**：需要極高視覺張力與前衛美學的展演宣傳。
*   **品牌視覺概念奠基**：為新品牌或新產品線尋求獨特的視覺哲學（Aesthetic Movement）。
*   **書籍、唱片與藝術封面**：需要傳遞抽象意境、故事靈魂與精緻工藝感的封面創作。
*   **空間裝飾與靜態藝術品**：用於實體空間懸掛、展示的博物館級抽象畫作。
*   **概念性科學/系統圖表**：以嚴謹的幾何、重複圖樣與精密標記，將抽象的人類體驗或假想學門視覺化的藝術創作。

# 如何觸發

範例1：請為一場探討「時間與重力」的現代舞展演，設計一張充滿空間張力的極簡藝術海報。

範例2：我想為我的獨立環境音樂（Ambient Music）專輯創作一張封面，希望能體現出「類比冥想」般的呼吸感與紙張肌理。

範例3：請以「粗獷主義的喜悅（Brutalist Joy）」為設計哲學，為一間前衛建築師事務所創作一幅視覺藝術作品。

範例4：為「幾何沉默（Geometric Silence）」這個主題撰寫設計哲學，並為我生成一張極致精細、具有瑞士理性主義美學的 PDF 設計圖。

範例5：請根據「微觀植物的模組化生長規律」，創作出一個精緻得如同科學文獻插圖的系統化藝術畫布。

# 輸入內容

*   **核心主題或微小線索（Subtle reference）**：可以是具體的主題（如：某場活動、專輯名稱），或是非常抽象的概念（如：一種情緒、一段哲學思考）。
*   **風格與美學偏好**：例如極簡、粗獷主義、瑞士網格、自然有機等。
*   **期望載體**：海報、藝術畫作、書籍封面等定位。

# 輸出內容

每次觸發將會經歷兩步驟，並輸出以下內容：

1.  **設計哲學文檔（.md 檔案）**：
    *   為該視覺運動命名（1-2個詞彙）。
    *   4-6 段詩意且深刻的設計哲學宣言（闡述空間、色彩、尺度、排版與結構的關係），並反覆強調匠人精神與極致工藝。
2.  **畫布藝術作品（.pdf 或 .png 檔案）**：
    *   高達 90% 的精美視覺設計，至多 10% 的點綴性文字。
    *   具備極致的排版、無重疊的呼吸感邊界、高雅的字體選用與無懈可擊的細節打磨。

# 注意事項

*   **文字極簡化**：本 Skill 的核心是「視覺溝通」，文字僅作為視覺點綴或系統標記，不適合用來呈現大段的說明文字、列表或活動詳情。
*   **無重疊原則**：所有圖案、字體、視覺元素皆有嚴格的邊界與呼吸空間，絕不允許任何元素重疊或溢出畫布邊界。
*   **拒絕俗套**：產出的藝術風格偏向高端、抽象、嚴謹的設計美學，不會產生卡通、廉價插圖或常見的 AI 模板風格。

# 常見問題

**Q：這個 Skill 可以幫我做促銷傳單或放上很多字的文章排版嗎？**
A：不適合。本 Skill 的定位是創作「藝術品」與「大師級設計」，強調空間美學。過多的文字會破壞工藝感。如果需要放活動資訊，通常會以極其精緻、細微的 clinical typography（臨床風格排版）作為視覺點綴，而非直接平鋪直敘。

**Q：產出的設計哲學（.md）和畫布（.pdf/.png）有什麼關聯？**
A：設計哲學是靈魂與骨架，畫布是肉身與實踐。設計哲學中定義的「色彩、幾何、律動與匠人工藝」，會被 100% 忠實且精緻地具現化在畫布作品中。

**Q：我可以要求生成多頁的作品嗎？**
A：可以。在多頁請求下，本 Skill 會像製作高級茶几書（Coffee Table Book）一樣，讓後續頁面成為第一頁美學的延伸與獨特轉折，宛如在說一個優雅的視覺故事。
