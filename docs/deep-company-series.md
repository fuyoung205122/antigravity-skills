# 深度公司系列：8 篇长文拆一家公司

# 功能說明

為特定公司撰寫 8 篇深度長文系列，並以嚴格的事實核查標準確保內容的準確性，適合公開發表在公眾號、視頻號等渠道。

# 解決什麼問題

市面上充斥著大量存在事實錯誤、數據口徑不清、主觀臆斷及「偽精確」計算的財經文章。本 Skill 的核心價值在於提供一套經過「教科書級別」嚴格事實核查標準（寫前警惕、寫後掃描）所產出的公司深度研究系列文章，幫助讀者獲得準確、客觀、有洞見的分析，避免常見的誤導。

# 適合什麼情境

*   當您希望對一家公司進行極其深入、全面的研究，並將成果以高品質系列文章形式公開發表時。
*   需要產出約 12 萬字、包含 8 個獨立篇章的深度分析，涵蓋公司認知重置、護城河、利潤引擎、隱藏資產、時代影響、財務分析、管理層評估及估值判斷。
*   目標讀者是願意投入 90 分鐘甚至更長時間來深度理解一家公司的專業或高階個人投資者。
*   每篇文章需能獨立分享，但又貫穿一套連貫的價值投資判斷框架。

# 如何觸發

請直接提供您希望深度研究的公司名稱。

# 輸入內容

需要提供一個明確的**公司名稱**作為研究對象。

# 輸出內容

*   一個包含 8 篇獨立長文的深度公司系列研究報告（總計約 12 萬字），以及一篇 `00-系列說明.md` 作為目錄索引。
*   每篇文章都將嚴格遵守「價值投資框架」和「嚴苛事實核查」標準。
*   系列文章會從多個維度拆解公司，包括但不限於：認知重置、護城河、利潤引擎、隱藏資產、時代影響、財務分析、管理層評估及估值判斷。
*   內容旨在為讀者提供「教科書級別」的深度理解，而非替讀者做投資決策或股價預測。

# 注意事項

*   **不適用情境:** 本 Skill 不適用於單篇研報、季報點評或行業研究，請使用 `/investment-research`、`/earnings-review` 或 `/industry-research` 等相關 Skill。
*   **嚴苛的事實核查標準是核心:** 寫作前會警惕「偽精確」陷阱（如概率加權期望值、第三方測算 MAU/市佔率、線性外推等），並在修訂時進行 7 項強制檢查（如跨篇數字一致性、口徑標註、避免重複加計等）。
*   **寫作風格規範:** 採用直接、犀利、價值投資視角、不預設立場的語氣；禁用主觀絕對化詞彙（如「顯然」、「必然」）和流量黨比喻；標題會使用反差數字或反共識結論作為鉤子。
*   **不替讀者做決策:** 所有篇章末尾會明確註明「不構成投資建議」，不預測股價，也不提供加權期望年化報酬。
*   **不為判斷背書:** 不會引用「X 大佬也持有」等內容為文章判斷背書。
*   **彈性篇數:** 若某篇內容不足，Skill 可能會合併篇章或減少總篇數，不強求必須寫滿 8 篇。
*   **合規與隱私:** 僅使用公開資訊撰寫報告；不使用使用者個人資訊；推送前會進行隱私字段掃描。

# 常見問題

*   **Q: 這個 Skill 和一般的投資研報有什麼不同？**
    A: 最大的不同在於深度和公開發表導向。本 Skill 產出的是一個 8 篇、約 12 萬字的系列長文，目標是為「願意花 90 分鐘讀懂一家公司」的讀者提供教科書級別的深度分析，每篇獨立成文但貫穿一套完整框架。一般的投資研報通常篇幅較短，多為單篇，且目標讀者及發布渠道不同。
*   **Q: 我是否可以指定某些部分特別深入？**
    A: 系列篇章已預設涵蓋了關鍵的深度拆解維度（如護城河、財務、管理層等）。若有特定關注點，建議在觸發 Skill 時，在公司名稱後簡要說明。
*   **Q: 如果我對某個數據有疑問，可以要求修改嗎？**
    A: 可以。本 Skill 極度重視事實核查。在您提出修訂意見後，Skill 會先進行原始數據交叉驗證，並判斷修訂級別（硬錯誤、主觀化、顆粒度、不可靠），然後進行修改並同步聯動檢查。
*   **Q: 文章內容會包含投資建議或股價預測嗎？**
    A: 不會。文章旨在提供深入分析和決策框架，但不會替讀者做投資決策，也不會預測具體股價或給出「期望年化報酬」的加權值。
