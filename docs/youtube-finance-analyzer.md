# Skill 名稱
YouTube 財經與知識分析助手 (YouTube Finance & Knowledge Analyzer)

# 功能說明
自動分析 YouTube 影片內容，提供深度財經宏觀分析與結構化知識萃取。

# 解決什麼問題
解決使用者在觀看長篇財經或知識型影片時，耗費過多時間卻難以快速精鍊核心重點、分析市場影響力或轉化為結構化筆記的痛點。

# 適合什麼情境
*   觀看投資理財、股市分析影片時，需要快速理解對總體經濟的影響。
*   觀看觀念教學、書籍解讀影片，需要系統化的大綱與核心概念萃取。
*   研究市場趨勢，需要將影片內容轉化為專業報告。
*   在時間有限的情況下，想快速評估影片內容是否值得深看。

# 如何觸發
直接在對話中貼上 YouTube 連結即可啟動，無需額外指令。

範例1：直接貼上網址：`https://www.youtube.com/watch?v=xxxxxxx`
範例2：`幫我分析這部影片：[網址]`
範例3：`這部影片對於通膨的看法是什麼？ [網址]`
範例4：`整理這部教學影片的核心概念：[網址]`
範例5：`從這部財經影片中提取行動建議：[網址]`

# 輸入內容
*   一個或多個有效的 YouTube 影片 URL。
*   可選：與影片相關的特定問題或追問。

# 輸出內容
產出一份高品質的 Markdown 格式報告，內容包含：
1.  **影片基本資訊**：標題、來源等。
2.  **核心概念與知識點萃取**：結構化的內容摘要。
3.  **總體經濟與財務影響分析**：針對市場趨勢、利率、資產配置等維度的深度剖析（若適用）。
4.  **行動建議 / 投資洞察 / 總結**：具備參考價值的實作方向。

# 注意事項
*   本技能會自動判斷分析視角（財經、知識或兩者並用），不需額外確認。
*   輸出內容一律使用**繁體中文**。
*   分析品質取決於影片內容的清晰度與長度。
*   本技能提供的資訊僅供參考，不構成任何正式的投資建議。

# 常見問題
**Q：如果影片是非財經類的，也可以使用嗎？**
A：可以。系統會自動切換至 `youtube-knowledge-distiller` 模式，為您進行知識點的提取與結構化整理。

**Q：需要提供影片逐字稿嗎？**
A：不需要。技能會自動嘗試讀取影片的大綱、資訊與重點。
