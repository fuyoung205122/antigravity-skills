# Slack GIF Creator

# 功能說明

專為 Slack 優化的動態 GIF 製作工具，提供符合 Slack 規範的尺寸建議、動畫概念與自動化優化流程。

# 解決什麼問題

在為 Slack 製作自訂表情符號 (Emoji) 或頻道訊息 GIF 時，使用者常遇到檔案體積過大無法上傳、尺寸不符導致模糊、或是動畫效果生硬等問題。此 Skill 整合了 Slack 的技術規範（如 128x128 限制）與專業的動畫函式（如緩動效果、粒子噴發），確保產出的 GIF 既美觀又符合平台要求。

# 適合什麼情境

1. **自製 Slack Emoji**：需要精準的 128x128 尺寸與極小檔案體積。
2. **動態公告**：在 Slack 頻道中發布具有吸引力的 480x480 動態訊息。
3. **品牌動態化**：將現有的公司 Logo 或圖片加上旋轉、跳動、呼吸燈等效果。
4. **複雜動畫模擬**：需要粒子噴發（Explode）、震動（Shake）或淡入淡出（Fade）等專業視覺效果。

# 如何觸發

範例1：幫我做一個 128x128 的跳動星星 GIF，要在 Slack 當 Emoji 用。

範例2：我上傳了這張圖片，請幫我加上旋轉效果並優化成 Slack 格式。

範例3：製作一個帶有「完成！」文字的抖動動畫 GIF，背景要是漸層色的。

範例4：建立一個 480x480 的粒子噴發效果 GIF，用於慶祝專案上線。

範例5：把這張心形圖片加上呼吸燈（Pulse）效果，並確保檔案大小適合 Slack。

# 輸入內容

1. **基本需求**：動畫的主題（如：星星、圓形、文字）。
2. **尺寸偏好**：Emoji（128x128）或訊息圖（480x480）。
3. **動作類型**：如跳動、旋轉、震動、淡入淡出、粒子效果等。
4. **參考素材**：（選填）使用者上傳的圖片或 Logo。

# 輸出內容

1. **GIF 檔案**：一個經過 Slack 優化的 `.gif` 檔案。
2. **規格清單**：包含尺寸、FPS、顏色數、總時長等技術資訊。
3. **相容性驗證**：確認該檔案是否通過 Slack 的上傳規範檢查。

# 注意事項

* **時長限制**：為確保 Emoji 循環順暢，建議時長控制在 3 秒以內。
* **字體渲染**：不建議直接使用系統 Emoji 字體（跨平台相容性差），建議以 PIL 原生圖形繪製。
* **檔案大小與品質**：若需大幅縮小檔案（如優化為 Emoji），會透過減少顏色數（低至 48 色）或降低 FPS 來達成，這可能會導致畫質略微下降。

# 常見問題

**Q：為什麼輸出的 GIF 顏色看起來比較少？**
A：為了符合 Slack Emoji 的檔案大小限制，我們會自動將顏色數壓縮至 48-128 色之間。

**Q：我可以指定動畫的速度嗎？**
A：可以，您可以要求調整 FPS（建議範圍 10-30）或動畫的循環週期。

**Q：上傳的圖片可以變透明嗎？**
A：是的，我們支援 RGBA 處理，可以製作具有透明背景的 GIF 以適應 Slack 的深淺色模式。
