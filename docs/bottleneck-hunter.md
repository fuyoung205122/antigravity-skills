# Skill 名稱
供应链瓶颈猎手：AI驱动的全球产业链瓶颈套利

# 功能說明
AI驱动，深度拆解全球“超级趋势”下的产业链，识别未被市场充分定价的第二、第三层供应链瓶颈，挖掘潜在投资机会。

# 解決什麼問題
传统投资研究往往聚焦于已知龙头和热门赛道，忽略了物理供应链中不显眼但至关重要的“咽喉”环节。这些环节一旦断供便会影响整个行业，但由于其处于产业链的“更深层”，市场关注度和定价往往不足。本Skill旨在通过AI驱动的深入分析，从物理供应链的“咽喉”位置出发，寻找那些被忽视但具备巨大超额收益潜力的公司，解决传统投研的信息盲区和定价偏差问题。

# 適合什麼情境
*   识别宏观“超级趋势”下的潜在投资机会。
*   进行深度产业研究和供应链风险分析。
*   寻找市场关注度较低、但具备高爆发潜力的“隐形冠军”。
*   评估特定行业或技术未来发展的瓶颈限制。
*   作为投资组合多样化，寻找非共识投资标的。
*   企业进行战略规划时，评估自身供应链的脆弱环节。

# 如何觸發
提供至少 5 個實際範例。

範例1：`供应链瓶颈猎手，扫描AI基础设施建设的瓶颈机会。`
範例2：`查找能源转型中的供应链瓶颈套利。`
範例3：`帮我分析半导体再工业化的供应链瓶颈。`
範例4：`全球产业链瓶颈猎手，帮我寻找潜在投资标的。`
範例5：`国防现代化趋势下的供应链瓶颈有哪些？`
範例6：`启动供应链瓶颈猎手，关注太空经济趋势。`
範例7：`给我一份关于AI基础设施建设的供应链瓶颈分析报告。`

# 輸入內容
您需要提供一个明确的“超级趋势”名称，或者一个宽泛的指令，让Skill从其内部跟踪的清单中选择。

*   **指定一个“超级趋势”名称：** 例如“AI基础设施建设”、“能源转型”、“国防现代化”、“半导体再工业化”、“太空经济”。
*   **宽泛指令：** 例如“扫描全球产业链瓶颈”、“寻找潜在瓶颈套利机会”，此时Skill将从其内部跟踪的超级趋势清单中选择进行分析。

# 輸出內容
您将收到一份结构化、深入的瓶颈分析报告，可能包含以下内容：

*   **趋势验证报告：** 确认指定“超级趋势”的持续性、物理性、规模性和加速性。
*   **分层供应链物理拆解：** 将趋势拆解至Layer 0到Layer 4的物理实体，识别重点扫描区（Layer 2-3）。
*   **供应链瓶颈地图：** 详细列出S、A、B级瓶颈，包括环节名称、原因、主要供应商及近期变化。
*   **瓶颈机会排名表：** 综合评估识别出的公司，包括公司代码、市值、收入、PS/PE、瓶颈评级、市场份额、收入增速、信号强度及估值判断。
*   **一页纸摘要：** 对每个推荐标的提供公司名、代码、瓶颈定位、看多/看空逻辑、催化剂时间线、主要风险、关键财务数据和估值安全边际检验。
*   **行动建议：** 明确指出是“执行深入研究”、“加入观察名单”还是“暂不追踪”。
*   **（定时任务模式下）每日瓶颈猎手报告：** 若有新信号或明确标的，会生成包含最新瓶颈信息和潜在投资机会的报告。

# 注意事項
*   **本Skill不直接推荐股票，而是提供深入的瓶颈分析和潜在标的线索。** 最终投资决策需用户自行判断并承担风险。
*   **高度重视估值：** 即使瓶颈逻辑再强，若标的公司估值已严重透支（如PS > 30x，市值远超TAM等），Skill会明确警示“估值透支”，甚至降低信号强度，建议等待更合理的价格。
*   **瓶颈具有时效性：** 供应链瓶颈会随着时间、技术发展和产能扩张而解除，需持续关注最新动态。
*   **信息限制：** 报告数据主要来源于公开信息，可能存在时滞或不完全之处，对于数据不足的部分会明确标示“数据不足”。
*   **避免AI偏见：** Skill设计了反向验证机制来对抗常见的投资研究偏见（如龙头偏好、英文偏好、确认偏见等），但用户仍需保持批判性思维。
*   **物理优先原则：** 仅关注需要实际硬件、材料或设备建设的物理性瓶颈，不涉及纯软件升级或概念炒作。

# 常見問題
1.  **这个Skill会直接告诉我买哪只股票吗？**
    *   不会。本Skill的核心是识别供应链瓶颈和潜在的优质标的，提供详尽的分析报告，但最终的投资决策仍需您自行判断。它强调“问题比答案重要”，帮助您理解市场深层逻辑。
2.  **我没有指定趋势，它会怎么做？**
    *   如果您没有指定具体趋势，Skill将自动扫描其内部跟踪的“超级趋势清单”（如AI基础设施、能源转型等），寻找其中的供应链瓶颈和投资机会。
3.  **为什么不直接关注那些耳熟能详的龙头公司？**
    *   本Skill认为，市场对龙头公司和第一层瓶颈（如GPU、HBM）的定价已相对充分。真正的超额收益（alpha）往往存在于市场关注度较低的第二、第三层供应链环节中。
4.  **如果扫描后没有发现任何值得投资的机会怎么办？**
    *   在“每小时扫描模式”下，如果没有发现新的瓶颈信号或明确的投资标的，Skill将不会生成报告，或只生成一份“信号扫描”报告，明确告知本次扫描无新发现。它不会为了出报告而凑数。
5.  **估值很高的公司，即使在瓶颈环节也会被推荐吗？**
    *   不会。本Skill将估值视为一个“硬门槛”。即使公司处于S级瓶颈，如果其估值已严重透支（如PS > 30x且收入增速不足、市值远超TAM的20%），报告会明确标示“估值透支”，甚至降低信号强度，建议等待更合理的价格。
6.  **报告中的数据来源是什么？**
    *   所有数据均来源于公开信息，包括公司财报、行业报告、新闻媒体、分析师研究等。Skill会尽可能标注数据来源，并对估计值进行说明。
