# Skill 名稱

财报精读：一手资料深度解读

# 功能說明

深度解读公司财报，直接從原始一手資料（年報、季報、電話會紀要等）中提煉核心洞察和投資判斷。

# 解決什麼問題

克服傳統AI投研工具過度依賴二手信息（如新聞、研報摘要、數據網站）的局限。它通過直接解讀公司原始財報、業績電話會紀要等一手資料，避免信息篩選、時滯和語境缺失的問題，提供更真實、更及時的深度投資分析。旨在幫助用戶像巴菲特和李錄一樣，直接從原始資料中獲取投資智慧。

# 適合什麼情境

*   **資深投資者或研究員**，希望進行高質量、基於一手資料的投資決策。
*   需要**快速掌握一家公司最新財報的核心亮點、風險和管理層信號**。
*   對市場普遍觀點持保留態度，尋求**獨立、深度的財報解讀**。
*   希望**追蹤公司管理層承諾兌現情況、識別潛在風險和隱藏信息**。
*   需要為**長期投資或價值投資**提供堅實的分析基礎。

# 如何觸發

提供至少 5 個實際範例。

範例1：`财报精读 腾讯 2025Q4`

範例2：`财报精读 PDD 2025年报`

範例3：`财报精读 美团 最新`

範例4：`幫我深度分析一下蘋果公司最新的財報`

範例5：`請對阿里巴巴2024年第三季度財報進行精讀分析`

# 輸入內容

需要提供目標分析的**公司名稱**（中文或英文，需明確）以及需要分析的**財報週期**（例如：季度，如`2025Q4`；年份，如`2025年報`；或`最新`代表最近一期）。

# 輸出內容

會得到一份結構化的深度財報精讀報告，包含以下主要部分：

*   **一、核心數據速覽**：關鍵財務指標（收入、利潤、現金流、資產負債）及其YoY變化。
*   **二、本期最重要的3個變化**：對報告期內最顯著的業務或財務動態進行總結。
*   **三、管理層語氣與承諾追踪**：分析管理層在討論中的信號（坦誠、清晰、模糊、轉移、外部化）以及此前承諾的兌現情況。
*   **四、附註中的隱藏信息**：挖掘關聯交易、股權激勵、或有負債、會計政策變更等關鍵附註項。
*   **五、關鍵問題**：提煉業績電話會中分析師的尖銳問題及管理層回答質量。
*   **六、與投資論文的關係**（如有持倉）：分析財報對現有投資判斷的影響。
*   **七、結論**：明確指出財報是超預期/符合預期/低於預期，對投資論文的影響（強化/無影響/削弱/破裂），以及後續需要關注的催化劑和操作建議（加倉/持有/減倉）。

報告會以`.md`格式保存到`reports/`路徑下。

# 注意事項

*   **資料來源優先級**：本Skill優先獲取公司IR頁面、SEC EDGAR、港交所披露易、巨潮資訊網等**一手原始資料**。若無法獲取，將標註為"非原始來源"或"一手資料不足"，並可能降低部分分析的權重。
*   **數據驗證**：所有關鍵財務數據都將通過`tools/financial_rigor.py`進行交叉驗證和核對，確保準確性。若來自第三方彙總的數據兩源誤差超過1%，將進行標記。
*   **分析深度**：側重於李錄、巴菲特等投資大師的分析方法，強調管理層討論（MD&A）、附註挖掘和承諾追蹤。
*   **結論明確性**：報告結論部分力求明確，不提供模糊的“基本符合預期”等表述，旨在幫助用戶形成清晰的投資判斷。
*   **不支持**：目前不支持對非上市公司的財報分析（因為缺乏公開的一手資料）。

# 常見問題

**Q1: 這個Skill支持分析哪些市場的公司財報？**
A1: 主要支持**美股（SEC EDGAR）、港股（港交所披露易）和A股（巨潮資訊網）**的上市公司財報。

**Q2: 如果我查詢的公司沒有提供完整的一手財報資料怎麼辦？**
A2: 在無法獲取完整一手資料的情況下，Skill會規範性地使用第三方標準數據源（如Macrotrends, StockAnalysis, AASTOCKS, 东方财富等）進行拼湊，但會**明確標註"非原始財報，來自第三方彙總"**，並降低附註分析的權重。

**Q3: 財報精讀的結論是投資建議嗎？**
A3: 本Skill提供的是基於深度分析的**投資判斷傾向**（例如：強化/削弱投資論文，加倉/減倉建議），但**並非直接的投資建議**。用戶仍需結合自身風險偏好和獨立判斷進行最終決策。

**Q4: 我可以分析很早以前的財報嗎？**
A4: 可以。只要您能提供具體的公司名稱和精確的財報週期（例如“騰訊 2018Q3”或“蘋果 2020年報”），Skill會嘗試獲取並分析該期間的財報。

**Q5: 報告中提到的“非GAAP”指標是什麼意思？**
A5: 非GAAP（Non-GAAP）指標是非通用會計準則下的財務指標，通常排除了某些非經常性或非現金項目（如股權激勵費用、商譽減值等）。Skill會區分並關注GAAP和Non-GAAP指標，以提供更全面的視角。
