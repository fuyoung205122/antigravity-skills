# Skill 名稱

财报精读团队：四大师并行解读 + 公众号发布

# 功能說明

這是一個 AI Berkshire 技能，它利用四位知名投資大師（段永平、巴菲特、芒格、李錄）的獨特視角，對公司財報進行多維度的深度解析。整個流程模擬一個專業團隊：由四位大師并行提供研究洞見，Team Lead 負責綜合分析，隨後編輯 Agent 將研究報告潤色改寫為適合微信公眾號發布的文章，最終由讀者评审 Agent 從普通投資者視角把關內容質量，確保產出兼具深度與易讀性的財報分析文章。

# 解決什麼問題

本 Skill 旨在解決進行高品質財報分析時的兩個核心挑戰：

1.  **自己能看懂未來：** 提供四個不同且互補的視角（生意本質、財務質量、競爭變化、風險信號），確保分析師能從多角度、深層次地理解財報背後反映的公司實質，避免單一視角的偏頗。
2.  **讀者能看懂價值：** 將複雜專業的財報分析，轉化為普通投資者也能理解並從中獲益的公眾號文章。透過專業編輯潤色和讀者視角的質量把關，確保內容不僅深入，而且易於消化、富有啟發性，最終可以直接用於公開發布。

# 適合什麼情境

*   當你需要對**重要公司**的**關鍵財報**進行**極致深度**和**多角度**分析時。
*   當你的分析結果不僅用於個人投資決策，還需要**對外發布**（例如：在公眾號、部落格等平台分享）。
*   希望藉助 AI 模擬一個**頂級投資團隊**的協作模式，為財報分析提供最全面的視角。
*   在需要一份**高質量、可信賴且易於傳播**的財報分析報告時。

# 輸出內容

本 Skill 將會生成一套完整的財報分析報告文件，包含：

*   **最終公眾號文章：** 一篇經過專業編輯潤色和讀者審閱的完整公眾號文章 (`reports/{公司名}/{公司名}-earnings-{期間}.md`)，可以直接發布。
*   **四大师合成研究報告：** 一份供內部參考的深度研究報告初稿 (`reports/{公司名}/{公司名}-earnings-{期間}-研究底稿.md`)。
*   **各大师獨立分析報告：** 四位投資大師（段永平、巴菲特、芒格、李錄）各自視角的詳細分析報告。
*   **讀者评审報告：** 一份從普通投資者角度對公眾號文章提出的具體修改意見和優化建議。
*   原始資料獲取情況、分析進度以及工具使用的詳細記錄。

# 注意事項

*   **資料獲取優先級：** Skill 會盡一切可能獲取公司 IR 頁面、SEC EDGAR、港交所披露易等**一手原始資料**。若僅能獲取部分原文或第三方匯總，會明確標註「非原始來源」，並相應調整分析深度。
*   **數據嚴謹性：** 關鍵財務數據至少會透過**兩個來源交叉驗證**，並使用專門的財務工具進行嚴格驗算，確保報告的數據準確性。
*   **結論明確化：** 報告要求提供**明確的投資結論**，避免模糊不清或模棱兩可的表述。
*   **反面檢驗：** 每個積極的發現都將附帶**反面論據**或**潛在風險**的討論，以確保分析的全面性和客觀性。
*   **編輯核心：** 編輯 Agent 的目標是**提升專業內容的可讀性**，而非降低其專業深度。它會讓複雜的財務術語變得易懂，但不會犧牲分析的嚴謹性。
*   **嚴格的準出流程：** 最終的公眾號文章在發布前會經過**數據抽檢**和**嚴格的質量審核**。任何不符合要求的內容都將被打回修正，直到通過審核為止。
*   **Team Lead 價值：** Skill 的核心價值在於 Team Lead 對四位大師不同視角的**綜合判斷**，找出共識、揭示矛盾，而非簡單地拼接報告。

# 常見問題

**Q1: 這個 Skill 和 `/earnings-review` 有什麼區別？**
A1: `/earnings-review` 是一個單 Agent 的財報精讀 Skill，旨在快速概覽財報，通常只需要一個特定視角。而 `/earnings-team` (本 Skill) 則是一個多 Agent 團隊協作的深度精讀 Skill，提供四位大師的多維度分析，並進一步產出可直接發布的公眾號文章，適用於需要更深度、更全面的財報分析，且有發布需求的場景。

**Q2: 如果我提供的公司名或期間不準確，Skill 會怎麼處理？**
A2: Skill 會盡力解析您的請求。如果公司名不明確，可能會要求進一步澄清。如果期間不準確或無效，Skill 會嘗試尋找最近或最相關的財報。如果無法識別，可能會提示您提供更精確的資訊。

**Q3: 為什麼需要讀者评审這個環節？它是如何運作的？**
A3: 讀者评审 Agent 扮演一位普通投資者的角色，從可讀性、信息價值、可信度和行動指導性等維度對編輯完成的公眾號文章進行審閱。它的目的是確保專業研究成果能被廣泛的讀者群理解和接受，提出修改意見，避免文章"自說自話"，讓內容更具普適性和影響力。

**Q4: 我可以自訂哪個大師參與分析或修改他們的分析角度嗎？**
A4: 目前本 Skill 預設啟用四位大師的固定分析框架。雖然您無法直接自訂每個大師的分析視角，但 Team Lead 會根據您提供的 `$ARGUMENTS` 總體需求來引導各 Agent 的分析，並在最後的合成與定稿階段進行整合判斷。
