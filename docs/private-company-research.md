# 未上市公司研究：多Agent并行深度研究框架

对未上市公司进行团队化深度研究分析。

# 功能說明

对未上市公司进行团队化深度研究分析。

# 解決什麼問題

未上市公司信息稀缺、不对称，传统研究方法难以深入，本框架通过多Agent并行，从不同维度深度挖掘信息，拼凑公司真实价值。

# 適合什麼情境

*   需要研究特定未上市公司（如：蚂蚁集团、小红书、SpaceX、Stripe）的真实价值。
*   希望在信息有限的情况下，获得比市场估值更贴近生意本质的价值判断。
*   对未上市公司投资决策提供支持。

# 如何觸發

**範例1：**
對「小紅書」進行未上市公司深度研究。

**範例2：**
研究 SpaceX 的真實價值。

**範例3：**
我想了解 Stripe 的商業模式、財務狀況、競爭格局及真實價值。

**範例4：**
使用未上市公司研究框架分析「螞蟻集團」。

**範例5：**
對一個名為「XYZ新創公司」的未上市公司進行全方位研究。

# 輸入內容

需要研究的未上市公司名稱。

# 輸出內容

一份結構化、多維度的未上市公司研究報告，包含：
*   一句話結論
*   公司画像速覽
*   六維評分總表（商業模式、財務估值、行業競爭、風險治理、技術IP、替代數據）
*   關鍵數據拼圖（交叉驗證後）
*   信號一致性矩陣
*   各維度分析摘要
*   真實價值評估（生意本質、護城河、估值判斷）
*   投資論點（看多/看空）
*   風險矩陣
*   退出路徑評估
*   投資決策表
*   信息盲區地圖
*   持續追蹤清單
*   總結段落

# 注意事項

*   **資訊透明度**：未上市公司信息來源複雜，所有數據都需標註來源、時間和置信度。
*   **推算邏輯**：所有推算過程需完全透明，展示計算邏輯和假設。
*   **交叉驗證**：關鍵數據至少2個來源交叉驗證，衝突時需列出所有來源並說明判斷。
*   **反偏見原則**：警惕AI研究偏見，寧可留白也不用推測填滿報告。
*   **替代數據價值**：重視招聘、專利、訴訟、App數據等替代數據。
*   **耐心等待**：多Agent并行研究需要時間，會實時更新進度。
*   **實事求是**：如果信息不足以做出可靠判斷，應明確說明。

# 常見問題

**Q1：如果我想研究的未上市公司資訊非常少，這個框架還有效嗎？**
A1：是的，框架特別設計了「應對AI研究偏見」的原則，對於信息極度稀缺的公司，會切換為「第一性原理模式」，聚焦核心問題，不追求報告完整性，而是提取關鍵判斷依據。報告中會明確標註信息缺失，並給出相應的置信度。

**Q2：報告的估值是絕對準確的嗎？**
A2：未上市公司估值本身存在較大不確定性。本框架會使用多種估值方法（融資估值、可比公司、DCF、終局市值倒推、交易對標）進行交叉驗證，並給出一個「真實價值區間」（保守、合理、樂觀），而非單一精確數字。報告中會詳細說明估值判斷的依據和置信度。

**Q3：我如何知道這份報告的可信度？**
A3：報告強調數據來源透明化，所有數據都標註來源、時間和置信度（🟢高/🟡中/🔴低）。同時，六維度的評分、風險矩陣、信息盲區地圖等都旨在幫助使用者全面了解報告的依據和局限性。

**Q4：報告的輸出很長，我該如何快速抓住重點？**
A4：報告開頭會提供「一句話結論」和「公司画像速覽」，中間有「六維度的分析摘要」，報告末尾還有「投資決策表」和「總結段落」，這些都是幫助您快速掌握核心信息的環節。
