# 行业漏斗筛选

# 功能说明

从全市场扫描，逐层精选出 3 家具有投资价值的行业标的。

# 解决什么问题

当投资者面对海量股票和众多行业时，难以系统性地识别出真正具有长期投资潜力的公司。本 Skill 旨在提供一个结构化的、标准化的筛选流程，帮助用户从全市场范围，高效地聚焦到少数几家优质标的，节省研究时间和精力。

# 适合什么情境

*   当你对某个行业或投资方向（如“AI 算力”、“创新药”、“机器人”）感兴趣，想了解其中最值得关注的投资标的时。
*   你需要一个系统性的方法来筛选股票，而不是凭感觉或片面信息。
*   你想把有限的研究精力集中在少数几家头部公司上。
*   你需要清晰的、可追溯的投资决策过程，了解每一层筛选的依据。
*   作为 `industry-research` Skill 的补充，在了解行业全景后，进一步进行个股精选。

# 如何触发

範例1：
请对“新能源汽车电池”行业进行价值投资漏斗筛选。

範例2：
我想了解“半导体设备”领域的投资机会，请帮我用漏斗模型筛选出 3 家优质公司。

範例3：
执行“生物科技”领域的行业漏斗筛选。

範例4：
请对“高端制造”行业进行价值投资精选。

範例5：
我需要一份关于“人工智能应用”行业的投资标的漏斗报告。

# 輸入內容

需要提供一个**行业名称**或**投资方向**。

# 輸出內容

一份详细的行业漏斗筛选报告，包含：
*   **第一层**：全市场扫描结果 (30-60 家活跃、涨幅或市值靠前的公司)。
*   **第二层**：价值投资 5 条硬指标粗筛结果 (≤ 10 家)。
*   **第三层**：精细分析结果 (≤ 10 家，每家 300-500 字的结构化分析)。
*   **第四层**：四大师（段永平、巴菲特、芒格、李录）视角深度分析的 3 家终选标的 (每家 800-1200 字)。
*   **综合输出**：终选 3 家组合表、行业级 ETF 替代建议、整体行业位置判断、信息充分度自评、待更新数据点、资料来源清单。
*   **淘汰记录**：每层被淘汰公司的名单及淘汰理由。

# 注意事項

*   筛选过程会考虑 A 股、港股、美股以及国际市场。
*   会对未上市的“未来 IPO 候选”进行标注。
*   报告会注明数据的来源，并对估计值进行标注。
*   AI 会自觉避免常见的研究偏见（如龙头偏好、英文偏好等）。
*   最终输出报告会经过数据抽检的准出流程。

# 常見問題

**Q1: "漏斗式价值投资筛选"具体是指什么？**
A1: 这是一种从广阔的市场范围开始，通过一系列预设的、量化的硬性指标和定性分析标准，层层递进地排除不符合条件的股票，最终聚焦到少数几家最有投资价值的公司的方法。

**Q2: "四大师深度分析"是什么意思？**
A2: 这是指对最终选出的 3 家核心标的，分别从段永平（生意本质）、巴菲特（护城河）、芒格（风险与失败模式）、李录（文明级趋势定位）这四位投资大师的独特视角进行深度解读和分析，以获得更全面、更深入的投资洞察。

**Q3: 如果某个行业找不到 3 家符合条件的标的怎么办？**
A3: Skill 会优先保证研究质量，如果某个子赛道找不到 3 家足够好的公司，会宁可报告终选 2 家，或标明“1 家观察”，避免为凑数而牺牲标准。

**Q4: 报告中的“护城河”是什么？**
A4: 护城河是指公司能够抵御竞争对手、保持长期竞争优势的壁垒。报告中列出了 5 类常见的护城河：品牌/定价权、转换成本/用户黏性、网络效应、规模效应、技术/牌照/资源壁垒。

**Q5: "信息充分度自评"有什么用？**
A5: 这是 AI 对本次研究过程中，公司财务数据、估值数据、行业格局判断、管理层信息等方面的信息完整度和可信度进行的主观评估。A、B、C 三个等级，用于提示用户报告结论的可信度，以及哪些部分需要后续进一步核实。
