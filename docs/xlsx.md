# Skill 名稱

xlsx

# 功能說明

專業的試算表處理專家，負責建立、讀取、編輯、格式化及分析 Excel (.xlsx, .xlsm) 與 CSV/TSV 檔案。

# 解決什麼問題

解決了手動處理大量表格數據的低效、財務模型中公式錯誤的風險、混亂原始資料的清洗難題，以及需要符合專業標準（如特定顏色代碼與數字格式）的報表製作需求。

# 適合什麼情境

*   **財務建模**：建立符合行業標準的資產負債表、現金流量表或預算預測。
*   **資料清洗**：將格式混亂的 CSV 或原始數據轉換為結構清晰、帶有標題與格式的 Excel。
*   **報表自動化**：在現有範本中追加數據，並保持原有的設計與公式邏輯。
*   **格式轉換**：在不同的表格檔案格式（如 TSV 轉 XLSX）之間進行轉換。
*   **公式修復**：檢查並修正 Excel 檔案中的 `#REF!`、`#DIV/0!` 等公式錯誤。

# 如何觸發

範例1：「請幫我打開下載資料夾中的銷售資料.csv，將它轉換為 Excel 並加上總計公式。」

範例2：「建立一個 2024 年的財務預測模型，藍色字體代表假設輸入，黑色代表公式計算。」

範例3：「讀取『Q3營收.xlsx』，清理裡面重複的列，並幫我計算各產品線的成長率。」

範例4：「我有一個 JSON 數據，請幫我轉成帶有 Arial 字體和綠色分頁標籤的 Excel 報表。」

範例5：「檢查『損益表.xlsx』裡的公式，確保沒有硬編碼（Hardcode），並確保所有計算正確。」

# 輸入內容

*   **檔案來源**：.xlsx, .xlsm, .csv, .tsv 檔案路徑或內容。
*   **處理需求**：計算邏輯（公式）、格式要求（字體、顏色、邊框）、資料清理規則。
*   **原始數據**：文字、JSON 或其他表格形式的數據源。

# 輸出內容

*   **實體檔案**：功能完整的試算表檔案（主要為 .xlsx）。
*   **動態內容**：包含可自動重算的 Excel 公式（而非靜態數值）。
*   **專業格式**：符合財務標準的顏色標記、專業字體與正確的數字格式（如貨幣、百分比）。

# 注意事項

*   **檔案優先**：最終產出必須是試算表檔案，不支援 Word 或純 HTML 報告。
*   **公式原則**：嚴禁在 Python 中計算結果後硬編碼（Hardcode）到 Excel，必須使用 Excel 公式（如 `=SUM(A1:A10)`）以保持動態性。
*   **財務標準**：預設使用藍色表示輸入值，黑色表示公式，綠色表示跨表連結。
*   **公式重算**：所有含公式的檔案都會經過 `recalc.py` 驗證，確保開啟時無錯誤。

# 常見問題

*   **Q: 這個 Skill 可以處理 Google Sheets 嗎？**
    A: 不可以，此 Skill 專注於處理本地試算表檔案，不支援 Google Sheets API。
*   **Q: 產生的 Excel 檔案打開會顯示公式錯誤嗎？**
    A: 不會，Skill 會在存檔前自動執行重算與錯誤掃描，確保零公式錯誤（#REF!, #VALUE! 等）。
*   **Q: 我原本的檔案格式會被弄亂嗎？**
    A: 不會，當更新現有範本時，Skill 會嚴格遵循並保留原有的格式與設計慣例。
