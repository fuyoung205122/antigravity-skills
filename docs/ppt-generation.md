# PPT Generation Skill

# 功能說明

自動生成專業的 PowerPoint 簡報。

# 解決什麼問題

手動製作簡報費時費力，且難以維持一致的視覺風格。此 Skill 能自動化簡報的圖像生成與排版，大幅提升效率與品質。

# 適合什麼情境

*   產品發表會簡報
*   學術演講簡報
*   商業提案簡報
*   創意匯報簡報
*   學校報告簡報

# 如何觸發

範例1：
「幫我製作一個關於 AI 產品發布的簡報，大約 5-10 張投影片，風格是 glassmorphism。」

範例2：
「我需要一份關於近期市場趨勢的簡報，請使用 dark-premium 風格，總共 7 張投影片。」

範例3：
「請為我的新創公司生成一份 10 張投影片的簡報，以 gradient-modern 風格呈現，主題是『我們的服務』。」

範例4：
「製作一份關於設計理念的簡報，使用 neo-brutalist 風格，大約 8 張。」

範例5：
「生成一個關於區塊鏈技術的簡報，使用 3d-isometric 風格，並指定 16:9 的長寬比。」

# 輸入內容

*   **簡報主題/內容綱要：** 簡報的主題或各投影片的重點內容。
*   **投影片張數：** 希望生成的投影片總數（預設為 5-10 張）。
*   **簡報風格：** 從列表中選擇一種風格（例如：glassmorphism, dark-premium, gradient-modern, neo-brutalist, 3d-isometric, editorial, minimal-swiss, keynote）。
*   **長寬比：** 選擇標準 (16:9) 或經典 (4:3)。
*   （可選）**風格指南：** 對顏色、排版、圖像、字體等細節的具體要求。

# 輸出內容

*   一個 `.pptx` 格式的 PowerPoint 簡報檔案。
*   （可選）每個投影片的獨立圖片檔案。

# 注意事項

*   圖像生成是按順序進行的，後續投影片的圖像會以前一張為參考，以確保視覺一致性。請勿嘗試平行生成。
*   務必提供具體的風格和內容描述，以獲得最佳效果。
*   對於圖像生成，請使用英文提示詞。
*   風格指南越詳細，生成的簡報視覺效果越能符合預期。

# 常見問題

**Q1：如何確保不同投影片之間的視覺風格一致？**
A1：Skill 在生成後續投影片的圖像時，會以前一張已生成的投影片作為參考，以維持整體視覺風格的連貫性。

**Q2：我可以指定特定的顏色或字體嗎？**
A2：是的，在輸入內容中提供詳細的風格指南，包括顏色調色盤（使用 HEX 值）、字體風格和排版要求，可以幫助 Skill 更精確地生成符合您期望的簡報。

**Q3：如果生成的某張投影片不滿意，可以重新生成嗎？**
A3：原則上，Skill 會一次性生成整個簡報。若有特定投影片不滿意，可能需要修改輸入的內容綱要或風格指南，然後重新生成整個簡報，或手動調整。

**Q4：我可以使用非英文的簡報主題嗎？**
A4：使用者可以輸入中文主題，但 Skill 在生成圖像提示詞時會自動轉換為英文，以確保圖像生成器的最佳效能。
