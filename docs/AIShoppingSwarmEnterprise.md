# AI Shopping Swarm Enterprise v2.0

# 功能說明

這是一個企業級多 Agent 網購協作決策系統，透過拆解採購任務並啟動多個專業 AI 代理人，同步執行搜尋、比價、規格驗證與風險評估。

# 解決什麼問題

解決單一 AI 在網購決策時資訊來源單一、缺乏深度比價、無法識別虛假評價、忽視隱性成本（如關稅、運費）以及容易產生幻覺推薦不實商品的問題。

# 適合什麼情境

*   **高單價決策**：如購買筆電、攝影器材、家電等需要精密規格比對的產品。
*   **跨平台比價**：需要同時分析 Amazon、淘寶、PChome、momo 等海內外平台的到手價。
*   **專業設備採購**：企業辦公設備、伺服器或生產工具的協作評估。
*   **規避購物風險**：對特定品牌或平台的售後、假貨風險、保值率有高度要求的採購。

# 如何觸發

提供至少 5 個實際範例。

*   **範例1：** 我想買一台 5 萬元左右的剪片用筆電，請幫我分析 MacBook Pro 與同價位 Windows 筆電的性能與未來保值率。
*   **範例2：** 幫我搜尋目前全網最划算的 PS5 主機方案，包含各平台回饋、信用卡折扣與歷史價格趨勢。
*   **範例3：** 企業採購需求：需要採購 10 台人體工學椅，單價預算 1 萬 5，重視保固年限與散熱材質，排除無品牌代工貨。
*   **範例4：** 我想從 Amazon 買咖啡機寄回台灣，請幫我計算包含關稅、變壓器需求、運費後的實際成本與維修風險。
*   **範例5：** 分析 Dyson 最新款吹風機與其競品的評價，特別是針對「一星評價」中的耐用度問題進行深度調研。

# 輸入內容

為了確保決策精準，建議提供以下資料（若不完整，系統將啟動 Phase 0 引導補充）：
*   商品類型與名稱。
*   預算範圍與幣別。
*   國家/地區（計算運費與稅務）。
*   使用情境（如辦公、電競、家用）。
*   偏好與排斥的品牌。
*   特殊需求（如：是否接受二手、平輸品、必要規格限制）。

# 輸出內容

您將得到一份極其詳盡的決策報告，包含：
1.  **執行架構與狀態表**：展示哪些 Agent 正在處理您的需求。
2.  **證據矩陣 (Evidence Matrix)**：條列所有結論的資料來源與信心程度。
3.  **商品候選名單**：含型號、平台、以及計算後的「最終到手價」。
4.  **深度分析報告**：包含優缺點、風險評鑑 (Risk Score) 與未來保值性分析。
5.  **Top 3 最終推薦**：給出明確的第一名建議。
6.  **CEO 最終結論**：由首席採購官 Agent 給出「如果是我會買哪一個」的果斷建議。
7.  **監控計畫**：後續降價或庫存變動的追蹤建議。

# 注意事項

*   **信心門檻限制**：若數據不足導致信心指數 (Confidence Score) 低於 70%，系統將拒絕推薦商品，改為要求您補充資訊。
*   **嚴禁猜測**：所有推薦必須基於現有證據，若無最新價格或規格資料，將會如實告知。
*   **序列檢查**：系統必須先通過 Phase 0/1 的需求分析，才能進入平行搜尋階段。

# 常見問題

*   **Q: 為什麼系統一直叫我補充資料而不直接給連結？**
    *   A: 這是為了避免誤判。AI Shopping Swarm 秉持企業級嚴謹性，在未明確您的使用情境或預算前，隨意推薦會導致採購失敗。
*   **Q: 系統可以分析目前的信用卡回饋嗎？**
    *   A: 可以，Price Intelligence Agent 會嘗試抓取主流平台的支付活動與折扣方案，但建議以最終結帳畫面為準。
*   **Q: 能夠識別「假評價」嗎？**
    *   A: Review Research Agent 會執行 Review Fraud Detection，分析社群論壇（如 Reddit, PTT）與電商評論的異常特徵。
